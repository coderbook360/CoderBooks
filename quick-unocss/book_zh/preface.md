在前端开发的浪潮中，CSS 解决方案的演进从未停止。从手写原生 CSS，到 BEM、Sass 等工程化方法，再到 CSS-in-JS，我们一直在寻找表达样式、组织代码的最佳方式。直到原子化 CSS（Atomic CSS）的出现，特别是以 Tailwind CSS 为代表的框架，为我们开辟了一条新的道路——通过组合高度可复用的、单一职责的工具类，来快速构建用户界面。

然而，当项目变得庞大，当性能的瓶颈开始显现，当定制化的需求超越框架的约束时，我们不禁会问：有没有一种更极致、更灵活、更高效的可能？

**答案是肯定的，这就是 UnoCSS 诞生的原因。**

UnoCSS 不是对现有工具的简单模仿或改进，而是一次对原子化 CSS 的“重新构想”（Reimagine）。它是一个即时的、按需生成的原子化 CSS 引擎，其设计哲学根植于极致的性能和无限的灵活性。

**为什么你需要这本书？**

如果你曾对以下问题感到困惑，那么这本书就是为你准备的：

*   **性能焦虑**：厌倦了在大型项目中等待 CSS 构建的漫长时间？UnoCSS 的即时引擎将为你带来几乎瞬时的开发体验。
*   **框架束缚**：是否觉得现有框架的规则和设计系统限制了你的创造力？UnoCSS 的核心是完全无主见的（un-opinionated），它将所有功能都交由“预设”（Presets）来提供，让你能够从零开始，自由构建属于你自己的设计系统。
*   **冗余与臃肿**：担心最终打包的 CSS 文件包含了大量未使用的样式？UnoCSS 严格按需生成，确保每一行 CSS 都物尽其用。
*   **表达力不足**：希望用更简洁、更直观的方式来组织和编写样式？UnoCSS 的“属性化模式”（Attributify Mode）、“变体组”（Variant Groups）和“纯 CSS 图标”等创新特性，将极大地提升你的开发幸福感。

**我们将一起探索什么？**

本书将不仅仅是一本 API 手册。我们将从 UnoCSS 的设计哲学和核心思想出发，引导你完成一次从“入门”到“精通”的旅程：

*   **第一部分：基础与核心思想**。我们将理解 UnoCSS 为何与众不同，并亲手构建一个微型规则引擎，从源头掌握其工作原理。
*   **第二部分：核心功能深度剖析**。你将学会驾驭官方提供的强大预设，无论是兼容 Tailwind，还是使用属性化模式和海量图标，都将得心应手。
*   **第三部分：高级技巧与自定义**。我们将深入引擎的内部，学习如何创建自己的预设、编写代码转换器，将 UnoCSS 完全定制成你想要的样子。
*   **第四部分：生态集成与实战**。你将学会在 Vue、React、Svelte 等主流框架中无缝集成 UnoCSS，并掌握调试、迁移等宝贵的实战经验。
