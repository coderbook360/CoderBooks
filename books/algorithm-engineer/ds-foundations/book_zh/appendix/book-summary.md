## 常用时间复杂度

| 数据结构 | 查找 | 插入 | 删除 |
|----------|------|------|------|
| 数组 | O(n) | O(n) | O(n) |
| 链表 | O(n) | O(1) | O(1) |
| 哈希表 | O(1) | O(1) | O(1) |
| BST | O(h) | O(h) | O(h) |
| 堆 | O(n) | O(log n) | O(log n) |

## 解题思维导图

1. **理解问题**：输入输出、边界条件
2. **识别模式**：属于哪类问题
3. **选择数据结构**：根据操作需求
4. **设计算法**：暴力→优化
5. **编码实现**：注意边界处理
6. **测试验证**：特殊用例

## 刷题建议

- **按专题刷**：先掌握一类再进入下一类
- **先理解后记忆**：理解为什么这么做
- **手写代码**：不要只看答案
- **总结模板**：建立自己的代码库
- **定期复习**：遗忘曲线
