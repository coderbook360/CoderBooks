## 3. 内容要点

### 核心问题
给定一个 URL 路径（如 `/user/123/posts`），如何从路由配置中找到匹配的路由？

### 关键知识点
1. 精确匹配 vs 模糊匹配
2. 动态段（`:id`）的处理
3. 通配符（`*`）的处理
4. 匹配优先级问题
5. 贪婪匹配 vs 非贪婪匹配
