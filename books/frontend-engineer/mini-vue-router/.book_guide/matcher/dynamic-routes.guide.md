## 3. 内容要点

### 动态路由类型
- 基本参数：`:id`
- 可选参数：`:id?`
- 重复参数：`:chapters+`
- 可选重复：`:chapters*`
- 自定义正则：`:id(\\d+)`
- 捕获所有：`:pathMatch(.*)*`

### 关键知识点
1. 每种参数类型的正则转换
2. 参数约束的实现
3. 通配符的特殊处理
4. 404 路由的实现方式

### 实现要点

```typescript
// 参数修饰符到正则的转换
const modifierToRegex = {
  '': '[^/]+',          // :id
  '?': '[^/]*',         // :id?
  '+': '[^/]+(?:/[^/]+)*', // :id+
  '*': '(?:[^/]+(?:/[^/]+)*)?', // :id*
}
```
