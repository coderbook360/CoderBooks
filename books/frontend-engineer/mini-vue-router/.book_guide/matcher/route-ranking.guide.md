## 3. 内容要点

### 核心问题

### 关键知识点
1. 优先级问题的产生场景
2. 静态段 > 动态段 > 通配符
3. 路由评分（Score）算法
4. 排序规则详解
5. 显式优先级配置

### 代码示例
```typescript
// 路由评分示例
'/user/profile'     // score: [80, 80]  静态
'/user/:id'         // score: [80, 40]  静态 + 动态
'/user/:id?'        // score: [80, 20]  静态 + 可选动态
'/:pathMatch(.*)*'  // score: [-80]     通配符
```
