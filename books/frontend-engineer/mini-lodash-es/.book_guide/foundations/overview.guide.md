## 3. 内容要点

### 核心概念
- **Lodash 的定位**: JavaScript 工具函数库，提供一致性、模块化、高性能的 API
- **设计理念**: 函数式编程、不变性、组合优先
- **模块化策略**: 单独发布每个方法、按需引入、Tree Shaking 支持

### 关键知识点
- Lodash 的历史与 Underscore.js 的关系
- Lodash 源码目录结构（dist/、lib/、.internal/）
- lodash、lodash-es、lodash/fp 三种版本的区别
- Lodash 的构建与发布机制
- 模块依赖关系图谱
