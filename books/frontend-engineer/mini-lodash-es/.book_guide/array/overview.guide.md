## 3. 内容要点

### 数组方法分类
| 分类 | 方法示例 | 用途 |
|------|---------|------|
| 分块与展平 | chunk, flatten, flattenDeep | 改变数组结构 |
| 过滤与去重 | compact, uniq, uniqBy | 移除元素 |
| 查找与索引 | find, findIndex, indexOf | 定位元素 |
| 集合运算 | difference, intersection, union | 多数组操作 |
| 取值与切片 | head, last, take, drop | 获取子集 |
| 变换与组合 | zip, unzip, fromPairs | 结构变换 |
| 移除操作 | pull, pullAll, remove, without | 就地/副本移除 |

### 方法命名规律
| 后缀 | 含义 | 示例 |
|------|------|------|
| 无后缀 | 基础版本 | uniq, difference |
| By | 接受迭代器函数 | uniqBy, differenceBy |
| With | 接受比较器函数 | uniqWith, differenceWith |
| Right | 从右侧开始 | dropRight, takeRight |
| Last | 返回最后一个 | findLast, lastIndexOf |

### 不可变 vs 可变
| 不可变（返回新数组） | 可变（修改原数组） |
|---------------------|-------------------|
| without, difference | pull, pullAll |
| uniq | remove |
| take, drop | fill |

## 5. 技术细节

### 源码参考
- array/ 目录下的所有文件

### 实现要点
- 大多数数组方法都依赖内部的 baseFlatten、baseUniq 等函数
- By 变体通过 baseIteratee 处理迭代器
- With 变体接受自定义比较函数

### 常见问题
- Q: 什么时候用 uniqBy，什么时候用 uniqWith？
- A: uniqBy 用于按某个属性去重，uniqWith 用于自定义复杂的比较逻辑
