## 3. 内容要点

### 核心概念
- **Lang 模块**: Lodash 中处理类型相关操作的模块
- **is* 方法**: 类型检测，返回布尔值
- **to* 方法**: 类型转换，返回目标类型值
- **clone* 方法**: 对象克隆，返回副本

### Lang 方法分类
| 分类 | 方法示例 | 用途 |
|------|---------|------|
| 基础类型判断 | isArray, isObject, isFunction | 检测值的基础类型 |
| 空值判断 | isNil, isNull, isUndefined, isEmpty | 检测空值状态 |
| 数值判断 | isNumber, isNaN, isFinite, isInteger | 检测数值类型和状态 |
| 相等性判断 | eq, isEqual, isEqualWith | 比较两个值是否相等 |
| 类型转换 | toArray, toString, toNumber, toInteger | 将值转换为目标类型 |
| 克隆 | clone, cloneDeep, cloneWith | 创建值的副本 |

### 常见问题
- Q: isObject 和 isPlainObject 有什么区别？
- A: isObject 包括函数和数组，isPlainObject 只判断普通对象
