# 高效编辑：操作符与文本对象组合

前面的章节介绍了 Vim 的语法模型。本章将深入实战，展示操作符与文本对象的各种组合如何应对真实编辑场景。

## 删除操作

`d`（delete）是最常用的操作符。

### 基础删除

| 命令 | 效果 |
|------|------|
| `x` | 删除光标下字符（等价于 `dl`） |
| `X` | 删除光标前字符（等价于 `dh`） |
| `dd` | 删除整行 |
| `D` | 删除到行尾（等价于 `d$`） |

### 配合动作

| 命令 | 效果 |
|------|------|
| `dw` | 删除到下一个词首 |
| `de` | 删除到词尾 |
| `db` | 删除到词首 |
| `d$` | 删除到行尾 |
| `d0` | 删除到行首 |
| `dj` | 删除当前行和下一行 |
| `dk` | 删除当前行和上一行 |
| `dgg` | 删除到文件首 |
| `dG` | 删除到文件尾 |

### 配合文本对象

| 命令 | 效果 |
|------|------|
| `diw` | 删除词 |
| `daw` | 删除词（含空格） |
| `di"` | 删除引号内内容 |
| `da"` | 删除整个字符串 |
| `di(` | 删除括号内内容 |
| `da(` | 删除整个括号 |
| `di{` | 删除花括号内内容 |
| `da{` | 删除整个花括号块 |
| `dit` | 删除 HTML 标签内容 |
| `dat` | 删除整个 HTML 标签 |
| `dip` | 删除段落 |

### 实战：删除函数参数

```javascript
function greet(name, age, country) {
```

想删除 `age, `：

1. 光标移动到 `age`
2. `daw` → 删除词，但不包括逗号
3. 手动删除逗号和空格

更好的方式：

1. 光标移动到 `,` 前的空格
2. `df,` → 删除到逗号（包含逗号）

结果：
```javascript
function greet(name, country) {
```

## 修改操作

`c`（change）= 删除 + 进入 Insert 模式。它是编辑代码最高频的操作。

### 与 d 对比

| d 命令 | c 命令 | 区别 |
|--------|--------|------|
| `dw` | `cw` | c 会进入 Insert 模式 |
| `diw` | `ciw` | c 删除后立即可以输入 |
| `dd` | `cc` | c 删除整行后停在原位，可以输入 |

### 常用组合

| 命令 | 场景 |
|------|------|
| `ciw` | 修改变量名 |
| `ci"` | 修改字符串内容 |
| `ci(` | 修改函数参数 |
| `ci{` | 修改函数体 |
| `cit` | 修改 HTML 标签内容 |
| `C` | 修改到行尾 |
| `cc` | 修改整行 |

### 实战：重命名变量

```javascript
const userName = getUserName();
console.log(userName);
```

想把 `userName` 改成 `fullName`：

1. 光标在 `userName` 上
2. `ciw` → 删除 `userName`，进入 Insert 模式
3. 输入 `fullName`
4. `Esc` 返回 Normal 模式
5. 移动到下一个 `userName`
6. `.` → 重复上次修改

`.` 命令（点命令）会在下一章详细讲解。它是 Vim 效率的核心。

### 实战：修改字符串

```javascript
const message = "Hello World";
```

想把 `"Hello World"` 改成 `"Hi Vim"`：

1. 光标在字符串内任意位置
2. `ci"` → 清空引号内容，进入 Insert 模式
3. 输入 `Hi Vim`
4. `Esc`

结果：
```javascript
const message = "Hi Vim";
```

注意：`ci"` 不管光标在引号内哪个位置，都会选中整个引号内容。这就是文本对象的威力——你不需要先选中，直接说"修改引号内"就行。

## 复制操作

`y`（yank）是 Vim 的复制命令。为什么叫 yank 而不叫 copy？因为 `c` 已经被 change 用了。

### 常用组合

| 命令 | 效果 |
|------|------|
| `yy` | 复制整行 |
| `Y` | 复制整行（与 `yy` 相同） |
| `yw` | 复制到下一个词首 |
| `yiw` | 复制当前词 |
| `yi"` | 复制引号内内容 |
| `yip` | 复制段落 |
| `y$` | 复制到行尾 |

### 粘贴

| 命令 | 效果 |
|------|------|
| `p` | 在光标后粘贴 |
| `P` | 在光标前粘贴 |

对于行级复制（如 `yy`）：
- `p` 粘贴到下一行
- `P` 粘贴到上一行

### 实战：复制函数

```javascript
function add(a, b) {
  return a + b;
}
```

想复制整个函数：

1. 光标在函数内任意位置
2. `va{` → 选中整个花括号块
3. 发现没选中 function 声明行...

更好的方式：

1. 移动到 `function` 行
2. `V` → 进入 Visual Line 模式
3. `}` → 扩展选择到函数结束
4. `y` → 复制

或者使用 `vap`（Visual around paragraph）如果函数是独立段落。

## 缩进操作

`>` 和 `<` 控制缩进：

| 命令 | 效果 |
|------|------|
| `>>` | 当前行增加缩进 |
| `<<` | 当前行减少缩进 |
| `>ip` | 段落增加缩进 |
| `<ip` | 段落减少缩进 |
| `>{` | 到上个段落增加缩进 |
| `>i{` | 花括号内增加缩进 |

### 配合 Visual 模式

选中多行后：
- `>` → 所有选中行增加缩进
- `<` → 所有选中行减少缩进

这在调整代码块缩进时非常实用。

### 自动格式化

`=` 自动格式化缩进：

| 命令 | 效果 |
|------|------|
| `==` | 格式化当前行 |
| `=ip` | 格式化段落 |
| `=i{` | 格式化花括号内 |
| `gg=G` | 格式化整个文件 |

## 大小写转换

| 命令 | 效果 |
|------|------|
| `~` | 切换光标下字符大小写 |
| `g~iw` | 切换词大小写 |
| `gUiw` | 词转大写 |
| `guiw` | 词转小写 |
| `gUU` | 当前行转大写 |
| `guu` | 当前行转小写 |

### 实战：常量转换

```javascript
const maxRetryCount = 3;
```

想改成常量风格 `MAX_RETRY_COUNT`：

1. 光标在 `maxRetryCount`
2. `gUiw` → 转大写 `MAXRETRYCOUNT`
3. 手动添加下划线（这个场景用插件更方便）

## 组合技巧

### 数字倍增

几乎所有命令都可以加数字前缀：

| 命令 | 效果 |
|------|------|
| `3dd` | 删除 3 行 |
| `5yy` | 复制 5 行 |
| `2dw` | 删除 2 个词 |
| `3>>` | 3 行增加缩进 |

### 操作符等待状态

按下操作符后，Vim 进入"等待状态"，等待动作或文本对象。

这时你可以：
- 继续输入动作/文本对象完成命令
- 按 `Esc` 取消

状态栏会显示你已输入的内容，帮助你确认。

### 可视化选择后操作

如果你不确定操作范围，可以先用 Visual 模式选中，再操作：

1. `v` → 进入 Visual 模式
2. 用移动命令扩展选择
3. 按操作符（`d`、`c`、`y`、`>`等）

例如 `viwd` 等价于 `diw`。区别是 Visual 模式让你能看到选中了什么。
