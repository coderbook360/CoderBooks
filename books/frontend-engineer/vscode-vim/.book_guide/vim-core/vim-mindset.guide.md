## 3. 内容要点
### 核心概念（必须全部讲解）
- **Vim 编辑语法**: `操作符 + 动作/文本对象`
  - 例如：`d`(delete) + `w`(word) = `dw`（删除一个词）
  - 例如：`c`(change) + `i`(`inside`) + `"`(引号) = `ci"`（修改引号内内容）

- **三大组成部分**:
  1. **操作符**（Operator）: d, c, y, v, >, <, =
  2. **动作**（Motion）: w, b, e, $, 0, gg, G
  3. **文本对象**（Text Object）: iw, aw, i", a", i(, a(, it, at

- **为什么 Vim 高效**:
  - 组合性：10 个操作符 × 10 个文本对象 = 100 种操作
  - 可预测性：理解语法后可以"说出"任何操作
  - 可重复性：`.` 命令重复上次操作

### 关键知识点
- **操作符列表**:
  - `d`: delete（删除）
  - `c`: change（修改，删除后进入 Insert 模式）
  - `y`: yank（复制）
  - `v`: visual（进入 Visual 模式选择）
  - `>`: 缩进
  - `<`: 反缩进
  - `=`: 自动格式化

- **文本对象示例**:
  - `iw` / `aw`: inner word / a word（含空格）
  - `i"` / `a"`: 引号内 / 含引号
  - `i(` / `a(`: 括号内 / 含括号
  - `it` / `at`: HTML 标签内 / 含标签
  - `ip` / `ap`: 段落内 / 含段落

- **实战示例**:
  ```
  原文：const message = "Hello World";
  
  ci"  → 修改引号内内容
  结果：const message = "|";  (光标在引号内)
  
  da"  → 删除整个字符串（含引号）
  结果：const message = ;
  
  yiw  → 复制当前单词（在 message 上）
  ciw  → 修改当前单词
  ```

## 5. 技术细节
- **完整组合示例表**:
  | 操作 | 命令 | 效果 |
  |------|------|------|
  | 删除一个词 | `dw` | delete word |
  | 删除到行尾 | `d$` | delete to end |
  | 修改引号内 | `ci"` | change inside " |
  | 复制括号内 | `yi(` | yank inside ( |
  | 删除整个函数 | `daf` | delete a function (需要插件) |
  | 修改标签内 | `cit` | change inside tag |
  | 选择段落 | `vip` | visual inner paragraph |
  | 格式化函数 | `=af` | format a function |

- **常见误区**:
  - `dw` vs `de` 的区别（是否包含空格）
  - `iw` vs `aw` 的区别（inner vs around）
  - 操作符必须在 Normal 模式下使用

## 10. 效率提升承诺
- 理解本章后，编辑操作思考时间减少 **70%**

**实战对比**:
```
任务：修改 "Hello World" 为 "Hi Vim"

传统方式：
1. 鼠标双击选中 "Hello"
2. 输入 "Hi"
3. 鼠标双击选中 "World"
4. 输入 "Vim"
时间：约 8-10 秒

Vim 方式：
1. ciw Hi <Esc>  (修改第一个词)
2. w              (移到下个词)
3. ciw Vim <Esc>  (修改第二个词)
时间：约 2-3 秒

或者：
1. ct" Hi<Space>Vim  (修改到引号内所有内容)
时间：约 1 秒
```
