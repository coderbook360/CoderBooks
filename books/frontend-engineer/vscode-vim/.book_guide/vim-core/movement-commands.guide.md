## 3. 内容要点
### 核心移动命令
**字符级移动**:
- `h/j/k/l`: 左/下/上/右
- `数字 + 动作`: 如 `5j`（向下5行）

**词级移动**:
- `w`: 下个词首
- `b`: 上个词首
- `e`: 词尾
- `W/B/E`: 按空格分隔的词（WORD）

**行内移动**:
- `0`: 行首
- `^`: 第一个非空字符
- `$`: 行尾
- `f{char}`: 跳到字符
- `t{char}`: 跳到字符前
- `;` / `,`: 重复 f/t

**跨行移动**:
- `gg`: 文件首
- `G`: 文件尾
- `{number}G`: 跳到第 N 行
- `Ctrl+D/U`: 半页上下
- `Ctrl+F/B`: 全页上下
- `{` / `}`: 段落间跳转

**屏幕相对移动**:
- `H`: 屏幕顶部
- `M`: 屏幕中间
- `L`: 屏幕底部
- `zz`: 光标所在行居中
