## 3. 内容要点（内容清单）
### 核心概念（必须全部讲解）
- **文件树聚焦**: `Ctrl+Shift+E` 切换到资源管理器视图
- **焦点管理**: 侧边栏焦点 vs 编辑器焦点的切换
- **上下文菜单**: VSCode 的上下文菜单键盘触发机制
- **Vim 导航映射**: 如何让文件树支持 `hjkl` 导航

### 关键知识点（必须全部覆盖）
- **基础导航键位**:
  - `↑/↓` 或 `j/k`: 上下移动
  - `→` 或 `l`: 展开文件夹/打开文件
  - `←` 或 `h`: 折叠文件夹/返回上级
  - `Enter`: 打开文件

- **文件操作键位**:
  - `a`: 新建文件（需自定义配置）
  - `Shift+a`: 新建文件夹（需自定义配置）
  - `r`: 重命名（或 `F2`）
  - `d`: 删除（需自定义配置）
  - `x`: 剪切
  - `c`: 复制
  - `v`: 粘贴

- **高级操作**:
  - `Shift+Alt+C`: 复制文件路径
  - `Shift+Alt+R`: 在资源管理器中显示
  - `/`: 搜索文件树（Filter 功能）
  - `Ctrl+0`: 快速回到文件树（从编辑器）

## 5. 技术细节（技术规范）
- **keybindings.json 配置示例**:
```json
[
  // 文件树导航（Vim 风格）
  {
    "key": "j",
    "command": "list.focusDown",
    "when": "explorerViewletFocus && !inputFocus"
  },
  {
    "key": "k",
    "command": "list.focusUp",
    "when": "explorerViewletFocus && !inputFocus"
  },
  {
    "key": "h",
    "command": "list.collapse",
    "when": "explorerViewletFocus && !inputFocus"
  },
  {
    "key": "l",
    "command": "list.expand",
    "when": "explorerViewletFocus && !inputFocus"
  },
  
  // 文件操作
  {
    "key": "a",
    "command": "explorer.newFile",
    "when": "explorerViewletFocus && !inputFocus"
  },
  {
    "key": "shift+a",
    "command": "explorer.newFolder",
    "when": "explorerViewletFocus && !inputFocus"
  },
  {
    "key": "d",
    "command": "moveFileToTrash",
    "when": "explorerViewletFocus && !inputFocus"
  },
  {
    "key": "r",
    "command": "renameFile",
    "when": "explorerViewletFocus && !inputFocus"
  },
  
  // 焦点切换
  {
    "key": "ctrl+0",
    "command": "workbench.files.action.focusFilesExplorer"
  },
  {
    "key": "ctrl+1",
    "command": "workbench.action.focusFirstEditorGroup"
  }
]
```

- **when 条件说明**:
  - `explorerViewletFocus`: 文件树获得焦点时
  - `!inputFocus`: 不在输入状态（避免覆盖文件名输入）
  - `filesExplorerFocus`: 文件资源管理器焦点

- **实现要点**: 
  - 文件树的键位不能与编辑器 Vim 键位冲突（通过 when 条件隔离）
  - 新建文件后自动聚焦到文件名输入框
  - 删除操作建议映射为"移动到回收站"而非永久删除

## 11. 实战演示脚本
提供一个完整的实战演示：
```
任务：在 src 目录下创建 components 文件夹，并创建 Button.tsx 和 Input.tsx

键盘流程：
1. Ctrl+Shift+E        # 聚焦文件树
2. /src                 # 搜索 src 文件夹
3. Enter                # 定位到 src
4. Shift+A              # 新建文件夹
5. 输入 "components"    # 文件夹名
6. Enter                # 确认
7. a                    # 新建文件
8. 输入 "Button.tsx"    # 文件名
9. Enter                # 确认并打开
10. Ctrl+0              # 回到文件树
11. a                   # 新建文件
12. 输入 "Input.tsx"    # 文件名
13. Enter               # 完成

总耗时：约 15 秒
鼠标操作：约 40-50 秒
```
