# 序言

## 经典代码的永恒价值

在React、Vue、Angular主导的今天，为什么还要学习jQuery源码？

答案很简单：**因为jQuery是JavaScript库设计的教科书。**

jQuery团队用十多年时间打磨出的代码，每一行都经过了无数真实场景的验证。链式调用、选择器引擎、事件委托、延迟对象这些设计思想深刻影响了后来的所有前端框架。

学习jQuery源码，你学到的不只是"jQuery怎么实现的"，而是：

- JavaScript闭包、原型链、作用域链的高级运用
- 如何设计优雅、易用、可扩展的API
- 如何在性能和灵活性之间做出正确的权衡
- 经过时间验证的代码组织和架构思想

这些能力，在你使用任何框架时都会受益。

## 本书的特点

**专注jQuery 3.7.x**

本书基于jQuery 3.7.x版本进行分析，不讨论旧版本的兼容代码。3.7.x面向现代浏览器，代码更加清晰简洁，是理解jQuery设计思想的最佳版本。

**源码驱动**

每个章节都从真实源码出发，逐行分析关键实现。不是泛泛而谈的概念讲解，而是深入到具体的函数和数据结构。

**图解辅助**

复杂的流程和数据结构都配有清晰的图解。从Sizzle选择器编译到事件系统的数据存储，让你一目了然。

**聚焦核心**

集中精力分析jQuery的核心模块：选择器、DOM操作、事件系统、Ajax、动画。这些是jQuery的精华所在。

## 你将学到什么

读完这本书，你将能够：

1. **理解jQuery的设计哲学**
   - 链式调用如何实现
   - 为什么这样组织代码结构
   - 如何在简洁和功能之间取舍

2. **掌握核心模块的实现原理**
   - Sizzle选择器引擎的编译与匹配
   - 事件委托的具体实现
   - Deferred/Promise的设计
   - 动画队列的调度机制

3. **提升JavaScript编程能力**
   - 闭包的高级应用
   - 原型链的灵活运用
   - 缓存与性能优化技巧

4. **学会阅读大型开源项目**
   - 建立源码阅读的方法论
   - 为阅读React、Vue源码打下基础

## 阅读准备

阅读本书，你需要：

- 熟练掌握JavaScript基础
- 理解闭包、原型链、this指向
- 有一定的DOM操作经验
- 最好使用过jQuery

如果某些概念不够熟悉，可以边读边查阅补充。

## 本书结构

本书共74章，分为十个部分：

**第一部分**是源码阅读的准备工作，包括源码结构和调试环境搭建。

**第二部分**分析核心架构：IIFE封装、jQuery对象、原型链、extend方法。

**第三部分**深入Sizzle选择器引擎，这是jQuery最精彩的部分之一。

**第四至五部分**分析DOM操作和属性样式处理。

**第六部分**剖析事件系统，包括绑定、委托、触发的完整实现。

**第七部分**分析Ajax系统和Deferred延迟对象。

**第八部分**讲解动画系统。

**第九部分**探讨Callbacks、Queue、Hooks等高级机制。

**第十部分**是实战总结，包括插件开发和设计模式。

建议按顺序阅读，后面的章节会依赖前面的知识。

## 开始你的源码之旅

阅读源码是成为高级工程师的必经之路。

jQuery是最适合入门的开源项目之一代码组织清晰，复杂度适中，设计思想经典。读懂jQuery，你将建立起阅读源码的信心和方法。
