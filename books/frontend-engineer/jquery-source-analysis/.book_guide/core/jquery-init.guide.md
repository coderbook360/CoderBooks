# 章节指导：jQuery函数与init构造器

## 章节定位
深入分析jQuery最核心的设计——`$()`函数的工作原理和init构造器的实现。

## 写作方向
- 聚焦工厂模式的设计思想
- 分析init如何处理不同类型的参数
- 忽略所有浏览器兼容性代码

## 学习目标
1. 理解`$()`为什么不需要new关键字
2. 掌握工厂模式在jQuery中的应用
3. 理解init构造器的参数处理逻辑
4. 学会分析"一个函数多种用途"的设计模式

## 核心内容要求

### 1. $()的本质
- jQuery函数的定义
- 返回new jQuery.fn.init()
- 工厂模式的实现

### 2. 为什么需要init
- 解决构造函数的问题
- init作为真正的构造器
- 原型链的连接技巧

### 3. init的参数处理
- 字符串选择器
- DOM元素
- 函数（DOM ready）
- HTML字符串
- jQuery对象
- 数组/类数组

### 4. 设计模式分析
- 工厂模式
- 参数重载
- 统一返回值
