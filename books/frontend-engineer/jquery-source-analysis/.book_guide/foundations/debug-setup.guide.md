# 章节指导：搭建调试环境

## 章节定位
本章是实战准备章节，帮助读者搭建可以调试jQuery源码的本地环境，为后续的深入学习提供工具支持。

## 学习目标
1. 成功克隆并构建jQuery源码
2. 配置Chrome DevTools调试环境
3. 掌握基本的断点调试技巧
4. 能够跟踪jQuery方法的执行流程

## 核心内容要求

### 1. 环境准备
- Node.js环境要求
- npm/yarn安装
- Git安装

### 2. 获取源码
- 克隆jQuery仓库
- 安装依赖
- 构建开发版本

### 3. 创建调试页面
- 创建HTML测试页面
- 引入jQuery源码
- 编写测试代码

### 4. Chrome DevTools调试
- 打开开发者工具
- 设置断点
- 单步调试
- 查看调用栈
- 监视变量

### 5. 常用调试技巧
- 条件断点
- 事件监听断点
- XHR断点
- 日志断点

### 6. VS Code调试（可选）
- 配置launch.json
- 浏览器调试插件

## 写作风格要求
- 步骤清晰，可操作性强
- 每一步都要给出预期结果
- 对可能遇到的问题给出解决方案

## 代码要求
- 给出完整的命令行指令
- 提供可运行的HTML测试页面
- 提供调试示例代码
