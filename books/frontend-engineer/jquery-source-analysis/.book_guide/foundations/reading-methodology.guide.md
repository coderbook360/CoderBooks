# 章节指导：源码阅读方法论

## 章节定位
本章是方法论章节，传授阅读复杂代码库的通用技巧，帮助读者建立正确的源码阅读心态和方法。

## 学习目标
1. 建立正确的源码阅读心态
2. 掌握自顶向下的阅读策略
3. 学会提炼代码的核心脉络
4. 建立有效的笔记系统

## 核心内容要求

### 1. 阅读心态
- 不求一次读懂，允许多次迭代
- 先求理解主线，再深入细节
- 带着问题阅读，保持好奇心

### 2. 阅读策略
- 自顶向下：从入口开始，逐层深入
- 自底向上：从工具函数开始，构建基础
- 问题驱动：带着具体问题寻找答案

### 3. 核心技巧
- 识别代码的主线和支线
- 暂时忽略边界处理和兼容代码
- 关注函数签名和返回值
- 绘制调用关系图

### 4. 笔记方法
- 记录核心发现
- 绘制流程图和关系图
- 写下疑问和思考
- 定期回顾和整理

### 5. jQuery源码的阅读顺序建议
- 推荐的模块阅读顺序
- 每个模块的重点关注点

## 写作风格要求
- 实用导向，给出具体可操作的建议
- 用jQuery源码作为例子说明
- 传授通用的阅读技巧，而非jQuery特定的知识
