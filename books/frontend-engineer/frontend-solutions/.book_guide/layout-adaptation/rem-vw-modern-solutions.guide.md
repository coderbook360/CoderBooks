## 2. 学习目标

### 知识目标
- 理解 rem 单位的工作原理与计算方式
- 掌握 vw/vh 视口单位的特性与使用场景
- 理解 flexible.js 等 rem 方案的实现原理
- 了解 clamp() 等现代 CSS 函数在适配中的应用

### 技能目标
- 能够实现基于 rem 的移动端适配方案
- 能够使用 vw 单位构建响应式布局
- 能够使用 clamp() 实现流式字体和间距
- 能够根据项目需求选择合适的适配方案

## 3. 内容要点

### 核心概念

| 概念 | 解释要求 |
|-----|---------|
| **rem 单位** | 解释 rem 与 em 的区别，rem 相对于根元素 font-size 的计算方式 |
| **vw/vh 单位** | 解释视口单位的定义，1vw = 视口宽度的 1% |
| **vmin/vmax** | 解释这两个单位的计算方式和使用场景 |
| **clamp() 函数** | 解释 clamp(min, preferred, max) 的工作原理 |
| **动态根字号** | 解释通过 JavaScript 动态设置根字号的原理 |

### 关键知识点

1. **rem 适配方案详解**
   - 动态计算根字号的原理
   - flexible.js 源码分析
   - postcss-pxtorem 等工具的使用
   - rem 方案的优势与局限

2. **vw 适配方案详解**
   - 纯 CSS vw 方案的实现
   - postcss-px-to-viewport 的使用
   - vw 方案与 rem 方案的对比
   - vw 的精度问题与解决

3. **现代 CSS 适配技术**
   - clamp() 实现流式排版
   - min()/max() 函数的应用
   - calc() 在适配中的应用
   - CSS 变量与适配的结合

4. **方案对比与选型**
   - rem vs vw 的核心差异
   - 各方案的兼容性对比
   - 不同项目类型的推荐方案
   - 混合使用策略

5. **实战：构建适配方案**
   - 设计稿尺寸标准化
   - 开发工作流配置
   - 边界情况处理
   - 与 UI 框架的配合
