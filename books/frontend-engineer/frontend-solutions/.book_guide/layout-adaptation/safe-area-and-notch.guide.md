## 2. 学习目标

### 知识目标
- 理解安全区域 (Safe Area) 的概念与由来
- 掌握 env() 和 constant() 函数的使用

- 能够正确配置 viewport-fit

- 能够兼容不支持安全区域的旧设备

## 3. 内容要点

### 核心概念

| 概念 | 解释要求 |
|-----|---------|
| **安全区域 (Safe Area)** | 解释安全区域的定义：不被刘海、圆角、Home 指示条遮挡的区域 |
| **env() 函数** | 解释环境变量函数，用于获取安全区域的值 |
| **constant() 函数** | 解释旧版 iOS 的写法，与 env() 的兼容关系 |
| **viewport-fit** | 解释 contain/cover/auto 三个值的区别 |
| **safe-area-inset-*** | 解释四个方向的安全区域内边距变量 |

### 关键知识点

1. **刘海屏的历史与影响**
   - iPhone X 引入刘海屏
   - 其他厂商的跟进与变种
   - 对 Web 布局的影响

2. **viewport-fit 详解**
   - auto：默认值，页面在安全区域内
   - contain：同 auto，确保页面完全可见
   - cover：页面延伸到全屏幕

3. **环境变量使用**
   - safe-area-inset-top
   - safe-area-inset-right
   - safe-area-inset-bottom
   - safe-area-inset-left
   - env() 与 constant() 的兼容写法

4. **常见适配场景**
   - 固定底部导航栏
   - 固定顶部导航栏
   - 全屏弹窗
   - 横屏模式

5. **最佳实践**
   - 使用 max() 函数提供降级值
   - 与固定定位的配合技巧
   - 与 flex 布局的结合
