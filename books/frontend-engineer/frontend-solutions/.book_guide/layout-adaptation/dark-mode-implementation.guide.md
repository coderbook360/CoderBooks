## 2. 学习目标

### 知识目标
- 理解暗黑模式的设计原理与用户价值
- 掌握 prefers-color-scheme 媒体查询的使用
- 理解 CSS 变量在主题切换中的作用
- 了解暗黑模式的色彩设计原则

### 技能目标
- 能够实现跟随系统的暗黑模式
- 能够实现用户可手动切换的主题系统
- 能够设计合理的暗黑模式色彩方案
- 能够处理图片和阴影在暗黑模式下的表现

## 3. 内容要点

### 核心概念

| 概念 | 解释要求 |
|-----|---------|
| **暗黑模式 (Dark Mode)** | 解释暗黑模式的定义、用户价值（护眼、省电） |
| **prefers-color-scheme** | 解释这个媒体特性的作用，light/dark/no-preference |
| **CSS 自定义属性** | 解释 CSS 变量在主题系统中的核心作用 |
| **color-scheme** | 解释 meta 标签和 CSS 属性的 color-scheme |
| **主题令牌 (Design Token)** | 解释语义化颜色变量的设计理念 |

### 关键知识点

1. **暗黑模式的价值与原理**
   - 用户偏好与系统设置
   - 护眼与 OLED 省电
   - 设计趋势与用户体验

2. **检测系统主题偏好**
   - prefers-color-scheme 媒体查询
   - window.matchMedia() API
   - 监听系统主题变化

3. **CSS 变量主题系统**
   - 定义语义化颜色变量
   - :root 与 [data-theme] 选择器
   - 主题变量的组织结构

4. **主题切换的实现**
   - 跟随系统 vs 用户手动
   - 主题偏好的持久化存储
   - 切换时的过渡动画

5. **暗黑模式的设计考量**
   - 颜色对比度与可访问性
   - 图片和阴影的处理
   - 品牌色的适配

6. **工程化实践**
   - 与 Tailwind/UnoCSS 的结合
   - 组件库的暗黑模式适配
   - 服务端渲染的闪烁问题
