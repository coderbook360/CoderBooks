## 3. 内容要点

### 核心概念

| 概念 | 解释要求 |
|-----|---------|
| **Long Task** | 解释超过 50ms 的主线程任务 |
| **主线程阻塞** | 解释 JavaScript 单线程对用户交互的影响 |
| **Long Task API** | 解释浏览器提供的长任务检测接口 |
| **任务拆分** | 解释将长任务分解为多个短任务的技术 |
| **时间切片 (Time Slicing)** | 解释分批执行任务的策略 |
