## 3. 内容要点

### 核心概念

| 概念 | 解释要求 |
|-----|---------|
| **真机调试** | 解释直接在真实设备上调试 |
| **USB 调试** | 解释通过 USB 连接设备调试 |
| **vConsole** | 解释移动端控制台工具 |
| **代理调试** | 解释通过代理抓包调试 |
| **远程调试** | 解释远程连接设备调试 |

### 关键知识点

1. **移动端调试的挑战**
   - 环境差异
   - 无法直接访问 DevTools
   - 网络环境复杂
   - 性能敏感

2. **Chrome 远程调试**
   - USB 调试开启
   - chrome://inspect
   - 端口转发
   - 调试 WebView

3. **Safari 远程调试**
   - 开发者模式开启
   - Web 检查器
   - 模拟器调试

4. **vConsole 使用**
   - 基本配置
   - 功能模块
   - 自定义面板
   - 按需加载

5. **代理调试工具**
   - Charles
   - Whistle
   - 抓包分析
   - Mock 数据

6. **特殊问题调试**
   - iOS Safari 兼容
   - 微信内置浏览器
   - 小程序调试
