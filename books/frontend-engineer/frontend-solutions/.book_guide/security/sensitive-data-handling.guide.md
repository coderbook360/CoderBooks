## 3. 内容要点

### 核心概念

| 概念 | 解释要求 |
|-----|---------|
| **敏感数据** | 解释需要特殊保护的数据类型 |
| **PII (个人身份信息)** | 解释可识别个人身份的信息 |
| **数据脱敏** | 解释隐藏或替换敏感数据的处理 |
| **前端加密** | 解释在前端对数据进行加密处理 |
| **安全存储** | 解释敏感数据的存储位置和方式 |

### 关键知识点

1. **敏感数据分类**
   - 个人身份信息
   - 金融信息
   - 认证凭证
   - 业务敏感数据

2. **存储安全**
   - localStorage 的风险
   - sessionStorage 的使用
   - Cookie 安全属性
   - 内存中的敏感数据

3. **数据脱敏**
   - 手机号脱敏
   - 身份证脱敏
   - 银行卡脱敏
   - 脱敏的时机

4. **传输安全**
   - HTTPS 的必要性
   - 敏感字段加密
   - 请求签名

5. **前端加密的讨论**
   - 前端加密的价值
   - 前端加密的局限
   - 合理的使用场景

6. **最佳实践**
   - 最小化敏感数据
   - 及时清理
   - 日志审计
   - 合规要求
