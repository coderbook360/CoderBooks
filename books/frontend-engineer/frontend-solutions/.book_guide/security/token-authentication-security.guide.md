## 3. 内容要点

### 核心概念

| 概念 | 解释要求 |
|-----|---------|
| **Token** | 解释用于身份认证的令牌 |
| **JWT (JSON Web Token)** | 解释自包含的结构化令牌格式 |
| **Access Token** | 解释用于访问资源的短期令牌 |
| **Refresh Token** | 解释用于刷新 Access Token 的长期令牌 |
| **Token 刷新** | 解释 Token 过期后的更新机制 |

### 关键知识点

1. **Token 认证原理**
   - Token vs Session
   - 无状态认证
   - 签名与验证

2. **JWT 结构解析**
   - Header
   - Payload
   - Signature
   - 常见声明

3. **Token 存储方案**
   - localStorage 风险
   - Cookie 存储
   - 内存存储
   - 混合方案

4. **Token 刷新机制**
   - 双 Token 方案
   - 滑动刷新
   - 刷新请求并发处理
   - 刷新失败处理

5. **安全风险与防护**
   - XSS 导致 Token 泄露
   - CSRF 与 Token
   - Token 劫持
   - 安全存储实践

6. **最佳实践**
   - Token 过期时间设置
   - 强制登出机制
   - Token 黑名单
   - 多设备管理
