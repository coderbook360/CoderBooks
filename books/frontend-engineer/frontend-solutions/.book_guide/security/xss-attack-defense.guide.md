### 技能目标
- 能够识别代码中的 XSS 漏洞

## 3. 内容要点

### 核心概念

| 概念 | 解释要求 |
|-----|---------|
| **XSS (Cross-Site Scripting)** | 解释跨站脚本攻击的定义 |
| **反射型 XSS** | 解释恶意脚本来自请求参数 |
| **存储型 XSS** | 解释恶意脚本存储在服务端 |
| **DOM 型 XSS** | 解释恶意脚本在客户端 DOM 操作中触发 |
| **输出编码** | 解释对输出内容进行转义处理 |

### 关键知识点

1. **XSS 攻击的本质**
   - 信任了不可信的输入
   - 将数据当作代码执行
   - 同源策略被绕过

2. **XSS 分类详解**
   - 反射型 XSS
   - 存储型 XSS
   - DOM 型 XSS
   - 各类型的危害程度

3. **攻击向量分析**
   - HTML 上下文
   - JavaScript 上下文
   - URL 上下文
   - CSS 上下文

4. **防御策略：输出编码**
   - HTML 实体编码
   - JavaScript 编码
   - URL 编码
   - CSS 编码
   - 上下文感知编码

5. **防御策略：输入验证**
   - 白名单验证
   - 黑名单的局限
   - 业务逻辑验证

6. **防御策略：CSP**
   - CSP 的作用
   - 内联脚本限制
   - 可信来源配置
   - 报告机制

7. **框架的 XSS 防护**
   - React 的默认编码
   - Vue 的模板编码
   - Angular 的安全机制
   - 危险 API 警示
