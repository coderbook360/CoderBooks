## 3. 内容要点

### 核心概念

| 概念 | 解释要求 |
|-----|---------|
| **CSP (Content Security Policy)** | 解释限制页面资源加载和执行的安全策略 |
| **指令 (Directive)** | 解释 CSP 中的各种限制规则 |
| **源表达式 (Source Expression)** | 解释定义可信来源的语法 |
| **Nonce** | 解释一次性随机值，用于白名单内联脚本 |
| **Hash** | 解释脚本内容的哈希值白名单 |

### 关键知识点

1. **CSP 的作用与价值**
   - XSS 的防御
   - 资源加载控制
   - 混合内容防护
   - 点击劫持防护

2. **CSP 指令详解**
   - default-src
   - script-src
   - style-src
   - img-src
   - connect-src
   - frame-src
   - 其他指令

3. **源表达式语法**
   - 'self'
   - 'unsafe-inline'
   - 'unsafe-eval'
   - 'nonce-xxx'
   - 'sha256-xxx'
   - URL 匹配

4. **Nonce 与 Hash 机制**
   - Nonce 的生成与使用
   - Hash 的计算与配置
   - 动态脚本的处理

5. **CSP 部署策略**
   - Report-Only 模式
   - 渐进式收紧
   - 监控与调整
   - 与 CDN 的配合

6. **报告机制**
   - report-uri 指令
   - report-to 指令
   - 报告内容分析
   - 报告处理系统
