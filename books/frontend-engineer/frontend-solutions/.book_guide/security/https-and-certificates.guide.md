## 3. 内容要点

### 核心概念

| 概念 | 解释要求 |
|-----|---------|
| **HTTPS** | 解释 HTTP over TLS 的安全传输协议 |
| **TLS (Transport Layer Security)** | 解释传输层安全协议 |
| **数字证书** | 解释服务器身份的证明文件 |
| **CA (Certificate Authority)** | 解释证书颁发机构 |
| **混合内容 (Mixed Content)** | 解释 HTTPS 页面加载 HTTP 资源 |

4. **混合内容问题**
   - 什么是混合内容

   - 解决方案
