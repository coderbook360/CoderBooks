## 3. 内容要点

### 核心概念

| 概念 | 解释要求 |
|-----|---------|
| **CSRF (Cross-Site Request Forgery)** | 解释跨站请求伪造的定义 |
| **Cookie 自动携带** | 解释浏览器对同域请求自动附带 Cookie |
| **CSRF Token** | 解释服务端生成的随机令牌 |
| **SameSite Cookie** | 解释限制 Cookie 跨站发送的属性 |
| **Referer 验证** | 解释通过请求来源验证合法性 |

### 关键知识点

1. **CSRF 攻击原理**
   - Cookie 的自动携带机制
   - 攻击的前置条件
   - 攻击流程演示

2. **CSRF 攻击方式**
   - GET 请求攻击
   - POST 请求攻击
   - 表单自动提交
   - 结合 XSS 的 CSRF

3. **防御策略一：CSRF Token**
   - Token 的生成与验证
   - Token 的存放位置
   - 双重 Cookie 验证
   - Token 刷新策略

4. **防御策略二：SameSite Cookie**
   - Strict 模式
   - Lax 模式
   - None 模式
   - 浏览器默认行为

5. **防御策略三：Referer/Origin 验证**
   - Referer 头检查
   - Origin 头检查
   - 验证的局限性

6. **最佳实践**
   - 多层防御
   - 关键操作二次确认
   - 敏感操作日志
   - 与前端框架集成
