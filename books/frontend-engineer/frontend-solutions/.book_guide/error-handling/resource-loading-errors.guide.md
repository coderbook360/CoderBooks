- 理解资源加载错误与 JavaScript 运行时错误的区别

- 了解 CDN 降级与资源重试策略

### 技能目标
- 能够监控各类资源的加载错误

## 3. 内容要点

### 核心概念

| 概念 | 解释要求 |
|-----|---------|
| **资源加载错误** | 解释 script, link, img, video 等资源加载失败 |
| **error 事件捕获** | 解释资源错误不冒泡，必须在捕获阶段监听 |
| **onerror 属性** | 解释元素上的 onerror 属性处理器 |
| **Performance API** | 解释如何通过性能 API 检测加载失败 |
| **资源降级** | 解释主资源失败时的备选方案 |

### 关键知识点

1. **资源错误的特殊性**
   - 资源错误不会冒泡到 window.onerror
   - 必须使用捕获阶段监听
   - 不同资源类型的错误表现

2. **捕获资源加载错误**
   - addEventListener('error', handler, true)
   - 区分 JavaScript 错误与资源错误
   - 获取失败资源的信息

3. **各类资源错误处理**
   - Script 脚本加载失败
   - CSS 样式表加载失败
   - 图片加载失败
   - 字体加载失败
   - 视频/音频加载失败

4. **关键资源的降级策略**
   - CDN 多源备份
   - 本地备份方案
   - 动态插入资源重试
   - 核心功能降级

5. **图片的特殊处理**
   - 占位图与默认图
   - 懒加载图片的错误处理
   - 图片格式降级 (WebP → JPEG)

6. **监控与上报**
   - 资源错误的数据采集
   - 区分偶发错误与系统性故障
   - 告警阈值设置
