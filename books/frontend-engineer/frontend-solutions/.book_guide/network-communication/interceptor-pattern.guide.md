## 3. 内容要点

### 核心概念

| 概念 | 解释要求 |
|-----|---------|
| **拦截器 (Interceptor)** | 解释在请求/响应过程中插入自定义逻辑的机制 |
| **请求拦截器** | 解释在请求发送前对配置进行修改 |
| **响应拦截器** | 解释在响应返回后对数据进行处理 |
| **Promise 链** | 解释多个拦截器通过 Promise 链串联执行 |
| **洋葱模型** | 解释拦截器的嵌套执行顺序 |

### 关键知识点

1. **拦截器的设计思想**
   - AOP (面向切面编程) 理念
   - 职责分离
   - 可插拔设计

2. **Promise 链实现**
   - 拦截器数组的组织
   - Promise.resolve().then() 链式调用
   - 拦截器顺序的影响

3. **Axios 拦截器源码分析**
   - 拦截器管理器实现
   - 拦截器链的构建
   - 同步/异步拦截器

4. **常用拦截器设计**
   - Token 注入拦截器
   - 响应数据格式化拦截器
   - 错误统一处理拦截器
   - 请求日志拦截器
   - 防重复请求拦截器

5. **拦截器的高级用法**
   - 条件性拦截
   - 拦截器的动态添加/移除
   - 拦截器优先级控制

6. **从零实现拦截器机制**
   - 拦截器管理器
   - Promise 链构建
   - 错误处理
