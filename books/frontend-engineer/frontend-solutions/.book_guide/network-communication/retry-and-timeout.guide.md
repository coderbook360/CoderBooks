## 3. 内容要点

### 核心概念

| 概念 | 解释要求 |
|-----|---------|
| **超时 (Timeout)** | 解释请求超过指定时间未响应则中断 |
| **重试 (Retry)** | 解释请求失败后自动重新发起请求 |
| **幂等性 (Idempotency)** | 解释多次执行产生相同结果的特性 |
| **指数退避 (Exponential Backoff)** | 解释重试间隔时间指数增长的策略 |
| **抖动 (Jitter)** | 解释在退避时间中加入随机因素 |

### 关键知识点

1. **超时控制的实现**
   - XMLHttpRequest 的 timeout 属性
   - Fetch 结合 AbortController
   - Promise.race 实现超时
   - 连接超时 vs 读取超时

2. **请求取消机制**
   - AbortController API
   - axios.CancelToken
   - 取消请求的时机

3. **重试策略设计**
   - 重试条件判断
   - 重试次数限制
   - 重试间隔策略
   - 什么情况不应该重试

4. **指数退避算法**
   - 基本公式
   - 抖动的作用
   - 最大退避时间
   - 代码实现

5. **幂等性与重试**
   - GET 请求的天然幂等
   - POST 请求的幂等改造
   - 请求 ID 去重
   - 后端幂等支持

6. **实战：完整的重试系统**
   - 配置化的重试策略
   - 重试日志与监控
   - 与拦截器的结合
