## 3. 内容要点

### 核心概念

| 概念 | 解释要求 |
|-----|---------|
| **虚拟列表 (Virtual List)** | 解释只渲染可视区域内元素的技术 |
| **可视区域 (Viewport)** | 解释用户可见的列表区域 |
| **缓冲区 (Buffer)** | 解释可视区域上下的额外渲染区域 |
| **滚动容器** | 解释产生滚动的父容器 |
| **列表项高度** | 解释定高与不定高的概念 |

### 关键知识点

1. **长列表的性能问题**
   - DOM 节点过多
   - 内存占用
   - 滚动卡顿
   - 首屏渲染慢

2. **虚拟列表核心原理**
   - 可视区域计算
   - 渲染范围确定
   - 定位与偏移
   - 占位元素

3. **定高虚拟列表实现**
   - 总高度计算
   - 起始索引计算
   - 结束索引计算
   - 偏移量计算

4. **不定高虚拟列表**
   - 预估高度
   - 动态测量
   - 缓存高度
   - 位置更新

5. **滚动优化**
   - 防抖与节流
   - RAF 优化
   - 缓冲区设计
   - 快速滚动处理

6. **开源库对比**
   - react-window
   - react-virtualized
   - vue-virtual-scroller
   - 各库特点
