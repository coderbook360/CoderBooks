## 3. 内容要点

### 核心概念

| 概念 | 解释要求 |
|-----|---------|
| **重排 (Reflow/Layout)** | 解释计算元素几何属性的过程 |
| **重绘 (Repaint)** | 解释将元素绘制到位图的过程 |
| **合成 (Composite)** | 解释合并图层生成最终画面 |
| **强制同步布局** | 解释 JS 读取几何属性触发的立即重排 |
| **合成层 (Compositing Layer)** | 解释独立渲染的图层 |

### 关键知识点

1. **渲染管道回顾**
   - Layout → Paint → Composite
   - 每个阶段的触发条件
   - 跳过阶段的优化

2. **触发重排的操作**
   - 改变元素几何属性
   - 读取某些属性触发强制重排
   - 常见触发列表

3. **触发重绘的操作**
   - 改变元素外观属性
   - 不影响布局的样式变化

4. **浏览器的优化策略**
   - 批量处理 DOM 操作
   - 队列刷新机制
   - 异步渲染

5. **优化策略**
   - 批量读写分离
   - 使用 transform 替代位移
   - 使用 requestAnimationFrame
   - 利用合成层
   - will-change 提示

6. **动画性能优化**
   - CSS 动画 vs JS 动画
   - 只动画 transform/opacity
   - GPU 加速的使用
