- **章节标题**: 图片优化的完整方案

- **所属部分**: 第五部分：性能优化实战

## 2. 学习目标

### 知识目标
- 理解图片对性能的影响
- 掌握各种图片格式的特点
- 了解响应式图片的实现方式
- 理解图片懒加载的原理

### 技能目标
- 能够选择合适的图片格式
- 能够实现响应式图片
- 能够实现图片懒加载
- 能够配置图片优化工具链

## 3. 内容要点

### 核心概念

| 概念 | 解释要求 |
|-----|---------|
| **图片格式** | 解释 JPEG, PNG, WebP, AVIF 等格式特点 |
| **响应式图片** | 解释根据设备提供合适尺寸的图片 |
| **懒加载 (Lazy Loading)** | 解释图片进入可视区域再加载 |
| **占位图 (Placeholder)** | 解释图片加载前的占位显示 |
| **渐进式图片** | 解释从模糊到清晰的加载方式 |

### 关键知识点

1. **图片对性能的影响**
   - 图片在页面资源中的占比
   - 对 LCP 的影响
   - 带宽与加载时间

2. **图片格式选择**
   - JPEG：照片类图片
   - PNG：透明图片
   - WebP：现代格式
   - AVIF：新一代格式
   - SVG：矢量图形
   - 格式选择决策

3. **响应式图片**
   - srcset 属性
   - sizes 属性
   - picture 元素
   - art direction

4. **图片懒加载**
   - loading="lazy" 原生支持
   - IntersectionObserver 实现
   - 占位图策略
   - 加载失败处理

5. **高级优化技巧**
   - 渐进式 JPEG
   - LQIP (低质量图片占位)
   - BlurHash
   - 图片 CDN

6. **工具与自动化**
   - 图片压缩工具
   - 构建时优化
   - CDN 动态处理
