## 3. 内容要点

### 核心概念

| 概念 | 解释要求 |
|-----|---------|
| **内存泄漏 (Memory Leak)** | 解释不再使用的内存无法被回收 |
| **垃圾回收 (GC)** | 解释自动回收不可达对象的机制 |
| **Heap Snapshot** | 解释内存堆快照，显示内存分布 |
| **Allocation Timeline** | 解释内存分配时间线 |
| **Detached DOM** | 解释从 DOM 树移除但仍被引用的节点 |

### 关键知识点

1. **JavaScript 内存管理**
   - 内存生命周期
   - 垃圾回收机制
   - 标记-清除算法
   - 引用计数

2. **常见内存泄漏模式**
   - 意外的全局变量
   - 未清理的定时器
   - 未移除的事件监听
   - 闭包引用
   - Detached DOM
   - Console 引用

3. **检测工具与方法**
   - Performance Monitor
   - Memory 面板
   - Heap Snapshot
   - Allocation Timeline

4. **Heap Snapshot 分析**
   - Shallow Size vs Retained Size
   - Object 视图
   - Comparison 视图
   - Containment 视图
   - Dominators 视图

5. **典型泄漏场景修复**
   - React/Vue 组件泄漏
   - SPA 路由切换泄漏
   - 大对象缓存
   - 事件总线订阅

6. **预防措施**
   - 代码规范
   - WeakMap/WeakSet
   - 生命周期管理
   - 定期检测
