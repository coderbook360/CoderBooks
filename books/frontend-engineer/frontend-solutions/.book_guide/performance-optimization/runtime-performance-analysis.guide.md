## 3. 内容要点

### 核心概念

| 概念 | 解释要求 |
|-----|---------|
| **运行时性能** | 解释页面加载后的交互和动画性能 |
| **帧率 (FPS)** | 解释每秒渲染的帧数，60fps 是流畅标准 |
| **火焰图 (Flame Chart)** | 解释函数调用堆栈的可视化 |
| **Long Task** | 解释超过 50ms 的主线程任务 |
| **Jank** | 解释掉帧导致的卡顿现象 |

### 关键知识点

1. **运行时性能指标**
   - FPS 与帧预算
   - 主线程占用
   - 内存使用
   - GPU 使用

2. **Performance 面板概览**
   - 录制与分析
   - 时间线区域
   - 主线程活动
   - 帧视图

3. **火焰图阅读技巧**
   - 调用堆栈解读
   - Self Time vs Total Time
   - 热点函数识别
   - 异步任务标记

4. **常见性能问题**
   - 长任务阻塞
   - 过度渲染
   - 内存抖动
   - 强制同步布局

5. **优化策略**
   - 减少主线程工作
   - 优化渲染流程
   - 减少 GC 压力
   - 利用空闲时间

6. **实战分析流程**
   - 问题重现
   - 录制分析
   - 定位瓶颈
   - 优化验证
