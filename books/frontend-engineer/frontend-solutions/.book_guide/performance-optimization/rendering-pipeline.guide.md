## 3. 内容要点

### 核心概念

| 概念 | 解释要求 |
|-----|---------|
| **渲染管道 (Rendering Pipeline)** | 解释从 HTML 到像素的完整流程 |
| **关键渲染路径 (Critical Rendering Path)** | 解释首屏渲染必须完成的最小工作 |
| **DOM 构建** | 解释 HTML 解析为 DOM 树的过程 |
| **CSSOM 构建** | 解释 CSS 解析为 CSSOM 的过程 |
| **渲染树 (Render Tree)** | 解释 DOM + CSSOM 合成渲染树 |

### 关键知识点

1. **渲染管道完整流程**
   - HTML 解析 → DOM
   - CSS 解析 → CSSOM
   - JavaScript 执行
   - 渲染树构建
   - 布局 (Layout)
   - 绘制 (Paint)
   - 合成 (Composite)

2. **关键渲染路径**
   - 关键资源识别
   - 关键路径长度
   - 关键字节数
   - 优化策略

3. **阻塞渲染的资源**
   - CSS 是阻塞渲染资源
   - JavaScript 的阻塞行为
   - async/defer 的作用
   - 字体加载的影响

4. **DOM 与 CSSOM**
   - 增量构建特性
   - 并行构建可能性
   - 相互依赖关系

5. **首屏优化策略**
   - 内联关键 CSS
   - 延迟非关键 CSS
   - 减少关键资源
   - 压缩关键资源

6. **工具与分析**
   - Lighthouse 分析
   - Performance 面板
   - 渲染阻塞分析
