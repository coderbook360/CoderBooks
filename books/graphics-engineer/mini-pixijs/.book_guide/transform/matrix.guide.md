## 3. 内容要点

### 核心概念（必须全部讲解）

#### 3.1 Matrix 类结构
```typescript
class Matrix {
  // 2x3 矩阵元素
  a: number;  // scaleX * cos(rotation)
  b: number;  // scaleX * sin(rotation)
  c: number;  // -scaleY * sin(rotation)
  d: number;  // scaleY * cos(rotation)
  tx: number; // translateX
  ty: number; // translateY
  
  // 矩阵表示（行主序）
  // | a  b  tx |   | x |   | a*x + c*y + tx |
  // | c  d  ty | × | y | = | b*x + d*y + ty |
  // | 0  0  1  |   | 1 |   | 1              |
}
```

#### 3.2 常见变换矩阵
| 变换类型 | 矩阵表示 |
|----------|----------|
| **平移** | `a=1, b=0, c=0, d=1, tx=dx, ty=dy` |
| **缩放** | `a=sx, b=0, c=0, d=sy, tx=0, ty=0` |
| **旋转** | `a=cosθ, b=sinθ, c=-sinθ, d=cosθ, tx=0, ty=0` |
| **斜切** | `a=1, b=tanα, c=tanβ, d=1, tx=0, ty=0` |

#### 3.3 矩阵乘法公式
```
C = A × B 表示先应用 B，再应用 A

c.a  = a.a * b.a + a.c * b.b
c.b  = a.b * b.a + a.d * b.b
c.c  = a.a * b.c + a.c * b.d
c.d  = a.b * b.c + a.d * b.d
c.tx = a.a * b.tx + a.c * b.ty + a.tx
c.ty = a.b * b.tx + a.d * b.ty + a.ty
```

#### 3.4 矩阵求逆
```
det = a * d - b * c  // 行列式
如果 det ≈ 0，矩阵不可逆

inv.a  =  d / det
inv.b  = -b / det
inv.c  = -c / det
inv.d  =  a / det
inv.tx = (c * ty - d * tx) / det
inv.ty = (b * tx - a * ty) / det
```
