## 3. 内容要点
### 核心概念（必须全部讲解）
- **点积定义**: $\mathbf{a} \cdot \mathbf{b} = a_x b_x + a_y b_y + a_z b_z$
- **点积几何意义**: $\mathbf{a} \cdot \mathbf{b} = \|\mathbf{a}\| \|\mathbf{b}\| \cos\theta$
- **点积的应用**: 计算夹角、判断方向、计算投影
- **叉积定义**: $\mathbf{a} \times \mathbf{b} = (a_y b_z - a_z b_y, a_z b_x - a_x b_z, a_x b_y - a_y b_x)$
- **叉积几何意义**: 垂直于两向量的新向量，长度等于平行四边形面积
- **右手法则**: 确定叉积方向的规则

### 关键知识点（必须全部覆盖）
- 点积的对称性：$\mathbf{a} \cdot \mathbf{b} = \mathbf{b} \cdot \mathbf{a}$
- 点积的正负号含义（锐角、直角、钝角）
- 单位向量点积直接等于夹角余弦值
- 叉积的反对称性：$\mathbf{a} \times \mathbf{b} = -(\mathbf{b} \times \mathbf{a})$
- 叉积仅在 3D 中有意义（2D 叉积是标量）
- 平行向量的叉积为零向量

## 5. 技术细节
- **实现要点**: 
  ```javascript
  dot(v) {
    return this.x * v.x + this.y * v.y + this.z * v.z;
  }
  
  cross(v) {
    return new Vector3(
      this.y * v.z - this.z * v.y,
      this.z * v.x - this.x * v.z,
      this.x * v.y - this.y * v.x
    );
  }
  ```
- **设计考虑**: 
  - 叉积返回新向量（符合数学定义）
  - 叉积的顺序敏感性
  - 右手坐标系的约定
- **边界情况**: 
  - 平行向量的点积和叉积
  - 零向量的点积和叉积
  - 数值精度问题（如判断垂直时的容差）
