## 5. 技术细节
- **实现要点**: 
  ```javascript
  // Lambert 漫反射
  function calculateDiffuse(normal, lightDir, lightColor, intensity) {
    const nDotL = Math.max(0, normal.dot(lightDir));
    return lightColor.multiplyScalar(intensity * nDotL);
  }
  
  // 运动更新
  function updatePosition(position, velocity, deltaTime) {
    return position.add(velocity.multiplyScalar(deltaTime));
  }
  
  // 点到平面距离
  function distanceToPlane(point, planeNormal, planePoint) {
    return point.sub(planePoint).dot(planeNormal);
  }
  ```
- **设计考虑**: 
  - 光照方向的归一化（必须是单位向量）
  - 时间步长在运动中的作用
  - 平面法向量的方向约定
- **边界情况**: 
  - 光照方向与法向量的夹角大于 90°（背面）
  - 零速度的物体
  - 点恰好在平面上
