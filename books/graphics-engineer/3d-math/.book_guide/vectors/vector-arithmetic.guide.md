# 章节写作指导：向量的加减与数乘

## 1. 章节信息
- **章节标题**: 向量的加减与数乘
- **文件名**: vectors/vector-arithmetic.md
- **所属部分**: 第一部分 - 向量运算
- **预计阅读时间**: 10-12分钟
- **难度等级**: 初级

## 2. 学习目标
### 知识目标
- 理解向量加法的几何意义（首尾相接）
- 理解向量减法的几何意义（两点之间的方向）
- 理解向量数乘的几何意义（缩放和反向）
- 掌握向量运算的计算规则

### 技能目标
- 能实现 Vector3 的 add() 方法
- 能实现 Vector3 的 sub() 方法
- 能实现 Vector3 的 multiplyScalar() 方法
- 理解如何在实际场景中应用这些运算

## 3. 内容要点
### 核心概念（必须全部讲解）
- **向量加法**: v1 + v2 的几何意义是"位移叠加"
- **向量减法**: v1 - v2 的几何意义是"从 v2 指向 v1 的向量"
- **向量数乘**: k * v 的几何意义是"方向不变，长度变为 k 倍"
- **运算的分量表示**: 逐分量计算

### 关键知识点（必须全部覆盖）
- 向量加法的平行四边形法则
- 向量减法在计算方向中的应用
- 负向量的含义（-v）
- 数乘为负时的反向
- 运算的交换律和结合律
- 实际应用：计算两点之间的方向向量

## 4. 写作要求
- **开篇方式**: 
  - 提出问题："如果一个物体先向右移动3个单位，再向上移动2个单位，最终位移是多少？"
  - 引出向量加法的必要性
- **结构组织**: 
  1. 向量加法（几何意义 → 计算规则 → 代码实现）
  2. 向量减法（几何意义 → 应用场景 → 代码实现）
  3. 向量数乘（几何意义 → 特殊情况 → 代码实现）
  4. 综合应用（实际例子）
- **代码示例**: 
  - 三个核心方法的实现（每个 5-8 行）
  - 使用示例（展示如何调用）
  - 代码演化：先展示最简单的实现，再讨论是否需要优化
- **图表需求**: 用文字和 ASCII 描述几何意义

## 5. 技术细节
- **实现要点**: 
  ```javascript
  add(v) {
    return new Vector3(
      this.x + v.x,
      this.y + v.y,
      this.z + v.z
    );
  }
  ```
- **设计考虑**: 
  - 是否返回新对象 vs 修改自身（不可变 vs 可变）
  - 参考 Three.js 的设计（返回 this 支持链式调用）
  - 是否需要静态方法（Vector3.add(v1, v2)）
- **边界情况**: 
  - 参数类型检查（是否需要）
  - 零向量的运算
