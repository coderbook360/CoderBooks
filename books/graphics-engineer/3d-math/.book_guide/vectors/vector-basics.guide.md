# 章节写作指导：向量的概念与表示

## 1. 章节信息
- **章节标题**: 向量的概念与表示
- **文件名**: vectors/vector-basics.md
- **所属部分**: 第一部分 - 向量运算
- **预计阅读时间**: 8-10分钟
- **难度等级**: 初级

## 2. 学习目标
### 知识目标
- 理解向量的几何意义（位置、方向、距离）
- 掌握向量的数学表示方法
- 理解 2D 向量与 3D 向量的区别与联系
- 理解向量的分量表示

### 技能目标
- 能用 JavaScript 实现 Vector3 类的基础结构
- 能够在代码中创建和表示向量
- 理解如何通过分量描述空间中的点和方向

## 3. 内容要点
### 核心概念（必须全部讲解）
- **向量的本质**: 既有大小又有方向的量（与标量的区别）
- **向量的几何意义**: 空间中的箭头，从起点到终点
- **向量的三种用途**: 表示位置（点）、表示方向、表示位移
- **向量的分量**: x, y, z 分量的含义和作用
- **行向量 vs 列向量**: 数学表示的两种形式

### 关键知识点（必须全部覆盖）
- 2D 向量回顾（从已知知识出发）
- 3D 向量的扩展（增加 z 轴）
- 坐标系的约定（右手坐标系）
- 向量的数学表示法：$(x, y, z)$ 或 $\begin{bmatrix} x \\ y \\ z \end{bmatrix}$
- Vector3 类的基础结构设计

## 4. 写作要求
- **开篇方式**: 从 2D Canvas 编程中的点和位移引入，自然过渡到向量概念
- **结构组织**: 
  1. 什么是向量（概念引入）
  2. 向量的几何意义（可视化理解）
  3. 向量的数学表示（数学形式）
  4. 从 2D 到 3D（扩展思维）
  5. 代码实现（Vector3 类骨架）
- **代码示例**: 
  - 创建 Vector3 类的基础结构（constructor）
  - 简单的向量创建示例
  - 代码量控制在 15-20 行以内
- **图表需求**: 用文字描述可视化（如"想象一个从原点指向 (1, 2, 3) 的箭头"）

## 5. 技术细节
- **实现要点**: 
  - 使用 ES6 class 语法
  - 属性：x, y, z（public）
  - 简单的 constructor(x=0, y=0, z=0)
  - toString() 方法用于调试
- **设计考虑**: 
  - 为什么使用 class 而不是对象字面量
  - 为什么默认值设为 0
  - 与 Three.js Vector3 设计对齐
- **边界情况**: 本章不涉及复杂边界情况，仅介绍基础概念

## 6. 衔接关系
- **前置知识**: Canvas 2D 坐标系统，基础 JavaScript
- **后续章节**: 第2章将学习向量的加减运算
- **过渡语句**: "现在我们知道了向量是什么，下一章将学习如何对向量进行运算"

## 7. 避免的问题
- ❌ 不要在第一章就讲向量运算（点积、叉积）
- ❌ 不要使用过于抽象的数学定义
- ❌ 不要讲复杂的线性代数理论
- ❌ 不要在代码中实现太多方法（只需骨架）
- ✅ 专注于"向量是什么"而不是"向量能做什么"
