## 3. 内容要点

### 核心概念

| 概念 | 解释要求 |
|------|---------|
| **变换矩阵** | 用 3x3 矩阵表示 2D 仿射变换 |
| **仿射变换** | 保持直线和平行关系的变换（平移、旋转、缩放、错切） |
| **齐次坐标** | 使用 (x, y, 1) 表示点，使平移可以用矩阵乘法表示 |
| **矩阵乘法** | 变换组合通过矩阵相乘实现 |

### 关键知识点

- Canvas 矩阵参数：a, b, c, d, e, f
- 矩阵的 3x3 形式与 6 参数的对应关系

- 矩阵乘法的非交换性（顺序影响结果）
- setTransform vs transform 的区别

### 边界与限制

- 矩阵参数顺序容易混淆
- 矩阵乘法顺序与直觉可能相反
- 某些变换矩阵不可逆（行列式为零）

## 5. 技术细节

### 实现要点

```javascript
// Canvas 变换矩阵
// | a  c  e |
// | b  d  f |
// | 0  0  1 |

// 点变换公式
// x' = a*x + c*y + e
// y' = b*x + d*y + f

// 单位矩阵（无变换）
// a=1, b=0, c=0, d=1, e=0, f=0
ctx.setTransform(1, 0, 0, 1, 0, 0);

// 平移矩阵
// a=1, b=0, c=0, d=1, e=tx, f=ty
ctx.setTransform(1, 0, 0, 1, tx, ty);

// 缩放矩阵
// a=sx, b=0, c=0, d=sy, e=0, f=0
ctx.setTransform(sx, 0, 0, sy, 0, 0);

// 旋转矩阵
// a=cos(θ), b=sin(θ), c=-sin(θ), d=cos(θ), e=0, f=0
const cos = Math.cos(angle);
const sin = Math.sin(angle);
ctx.setTransform(cos, sin, -sin, cos, 0, 0);

// transform 累积变换 vs setTransform 重置变换
ctx.transform(1, 0, 0, 1, 100, 100);  // 在当前基础上累加
ctx.setTransform(1, 0, 0, 1, 100, 100);  // 重置为指定变换

// 获取当前变换
const matrix = ctx.getTransform();
console.log(matrix.a, matrix.b, matrix.c, matrix.d, matrix.e, matrix.f);

// 重置变换
ctx.resetTransform();
// 等同于 ctx.setTransform(1, 0, 0, 1, 0, 0);

// 矩阵乘法（变换组合）
function multiplyMatrices(m1, m2) {
  return {
    a: m1.a * m2.a + m1.c * m2.b,
    b: m1.b * m2.a + m1.d * m2.b,
    c: m1.a * m2.c + m1.c * m2.d,
    d: m1.b * m2.c + m1.d * m2.d,
    e: m1.a * m2.e + m1.c * m2.f + m1.e,
    f: m1.b * m2.e + m1.d * m2.f + m1.f
  };
}
```

### 常见问题

| 问题 | 解决方案 |
|------|---------|
| 矩阵参数顺序混乱 | 记住是列优先：(a,b) 是第一列 |
| 变换结果与预期不符 | 使用 getTransform 调试当前矩阵 |
| 不知道 setTransform 和 transform 的区别 | setTransform 重置，transform 累积 |
| 矩阵乘法结果不对 | 注意乘法顺序，M1*M2 ≠ M2*M1 |
