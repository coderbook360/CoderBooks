## 3. 内容要点

### 核心概念

| 概念 | 解释要求 |
|------|---------|
| **坐标系变换** | Canvas 变换改变的是坐标系，而非图形本身 |
| **translate** | 平移坐标原点 |
| **rotate** | 旋转坐标系（绑原点） |
| **scale** | 缩放坐标系 |
| **变换累积** | 多次变换会累积效果 |

### 关键知识点

- translate(x, y) 的参数含义
- rotate(angle) 使用弧度制
- scale(x, y) 可以非等比例缩放
- 变换顺序的重要性
- 负缩放实现镜像
- 使用变换简化绑制代码

### 边界与限制

- 变换会影响所有后续绘制
- 变换不会影响已绘制的内容
- rotate 默认绕原点旋转
- 极端缩放值可能导致问题

## 5. 技术细节

### 实现要点

```javascript
// 平移
ctx.translate(100, 100);
// 之后所有绑制的 (0, 0) 实际是画布的 (100, 100)

// 旋转（默认绕原点）
ctx.rotate(Math.PI / 4);  // 旋转 45 度

// 缩放
ctx.scale(2, 2);  // 放大 2 倍
ctx.scale(-1, 1);  // 水平镜像
ctx.scale(1, -1);  // 垂直镜像

// 绕中心点旋转的标准模式
function drawRotatedRect(ctx, x, y, width, height, angle) {
  const centerX = x + width / 2;
  const centerY = y + height / 2;
  
  ctx.save();
  ctx.translate(centerX, centerY);  // 1. 平移到中心
  ctx.rotate(angle);                 // 2. 旋转
  ctx.translate(-centerX, -centerY); // 3. 平移回去
  ctx.fillRect(x, y, width, height); // 4. 绘制
  ctx.restore();
}

// 或更简洁的方式（在原点绑制）
function drawRotatedRect2(ctx, x, y, width, height, angle) {
  ctx.save();
  ctx.translate(x + width / 2, y + height / 2);
  ctx.rotate(angle);
  ctx.fillRect(-width / 2, -height / 2, width, height);
  ctx.restore();
}

// 变换顺序对比
// 先平移后旋转
ctx.translate(100, 100);
ctx.rotate(Math.PI / 4);
// vs 先旋转后平移
ctx.rotate(Math.PI / 4);
ctx.translate(100, 100);
// 结果完全不同！
```

### 常见问题

| 问题 | 解决方案 |
|------|---------|
| 图形旋转位置不对 | 理解绕原点旋转，使用平移组合 |
| 变换影响了其他图形 | 使用 save/restore 隔离变换 |
| 缩放后线宽也变了 | 正常行为，如需保持需除以缩放因子 |
| 变换累积导致混乱 | 每次变换前 save，绑制后 restore |
