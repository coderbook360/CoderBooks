## 5. 技术细节

### 实现要点

```javascript
// 鼠标坐标转换的标准方法
function getCanvasCoordinates(canvas, event) {
  const rect = canvas.getBoundingClientRect();
  const scaleX = canvas.width / rect.width;
  const scaleY = canvas.height / rect.height;
  
  return {
    x: (event.clientX - rect.left) * scaleX,
    y: (event.clientY - rect.top) * scaleY
  };
}

// 像素对齐技巧
// 绘制 1px 宽度的清晰线条
ctx.beginPath();
ctx.moveTo(100.5, 50.5);  // 使用 .5 偏移
ctx.lineTo(100.5, 150.5);
ctx.lineWidth = 1;
ctx.stroke();
```

### 常见问题

| 问题 | 解决方案 |
|------|---------|
| 1px 线条看起来是 2px 且模糊 | 使用 0.5 像素偏移 |
| 鼠标坐标与绘制位置不匹配 | 使用 getBoundingClientRect 计算偏移 |
| Canvas 缩放后坐标错误 | 考虑 scaleX/scaleY 缩放因子 |
