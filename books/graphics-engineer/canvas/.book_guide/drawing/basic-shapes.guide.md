## 5. 技术细节

### 实现要点

```javascript
// 角度转弧度
function degToRad(degrees) {
  return degrees * Math.PI / 180;
}

// 绘制圆形
function drawCircle(ctx, x, y, radius) {
  ctx.beginPath();
  ctx.arc(x, y, radius, 0, Math.PI * 2);
  ctx.fill(); // 或 ctx.stroke()
}

// 绘制扇形
function drawSector(ctx, x, y, radius, startAngle, endAngle) {
  ctx.beginPath();
  ctx.moveTo(x, y);
  ctx.arc(x, y, radius, degToRad(startAngle), degToRad(endAngle));
  ctx.closePath();
  ctx.fill();
}

// 绘制椭圆
function drawEllipse(ctx, x, y, radiusX, radiusY, rotation = 0) {
  ctx.beginPath();
  ctx.ellipse(x, y, radiusX, radiusY, rotation, 0, Math.PI * 2);
  ctx.fill();
}
```

### 常见问题

| 问题 | 解决方案 |
|------|---------|
| 圆形看起来被切掉一部分 | 检查起始/结束角度是否正确（应为 0 到 2π） |
| 弧形方向与预期相反 | 调整 counterclockwise 参数 |
| 椭圆不显示 | 检查浏览器兼容性，考虑使用 Polyfill |
