# 3D 场景管理与高级渲染

本书讲解现代 3D 引擎的核心技术，包括场景图、材质系统、光照系统和性能优化。

## 📚 本书定位

- **系列位置**：图形学系列 Book 4
- **前置知识**：Book 3 - WebGL 基础与渲染管线
- **后续学习**：Book 5 - Three.js 架构与核心实现
- **学习周期**：5-6 周

## 🎯 学习目标

1. 理解场景图的设计原理
2. 掌握相机系统和视锥体剔除
3. 深入理解 PBR 材质系统
4. 掌握多光源和阴影系统
5. 熟练应用性能优化技术
