# WebGL 基础与渲染管线

本书深入讲解 WebGL API 和 GPU 渲染原理，带你从零开始手写一个 WebGL 渲染器。

## 📚 本书定位

- **系列位置**：图形学系列 Book 3
- **前置知识**：Book 2 - 3D 图形学数学基础
- **后续学习**：Book 4 - 3D 场景管理与高级渲染
- **学习周期**：4-5 周

## 🎯 学习目标

1. 深入理解 WebGL API 和状态机模型
2. 掌握着色器编程（GLSL）
3. 理解 GPU 渲染管线的工作原理
4. 熟练使用缓冲区、纹理、深度测试等核心功能
5. 能够手写一个基础的 WebGL 渲染器
