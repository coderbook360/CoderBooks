# 图元装配与光栅化

> "光栅化是将连续几何图形转换为离散像素的过程。"

## 图元装配

### 图元类型

WebGL 支持三种基本图元类型：

```
点 (POINTS)              线 (LINES)              三角形 (TRIANGLES)
    •                    •─────•                     •
    •                    •─────•                    / \
    •                    •─────•                   •───•
```

### 绘制模式

```javascript
// 点
gl.drawArrays(gl.POINTS, 0, count);

// 线
gl.drawArrays(gl.LINES, 0, count);        // 每两个顶点一条线
gl.drawArrays(gl.LINE_STRIP, 0, count);   // 连续线段
gl.drawArrays(gl.LINE_LOOP, 0, count);    // 闭合线段

// 三角形
gl.drawArrays(gl.TRIANGLES, 0, count);        // 每三个顶点一个三角形
gl.drawArrays(gl.TRIANGLE_STRIP, 0, count);   // 三角形带
gl.drawArrays(gl.TRIANGLE_FAN, 0, count);     // 三角形扇
```

### 三角形拓扑

```
TRIANGLES:                  TRIANGLE_STRIP:          TRIANGLE_FAN:
v0─v2  v3─v5               v0──v2──v4               v0───v1
 \  |   \  |                \ | \ |                  |\ /|
  \ |    \ |                 \|  \|                  | X |
   v1     v4                 v1──v3                 v4──v3──v2

索引: [0,1,2, 3,4,5]       索引: [0,1,2,3,4]        索引: [0,1,2,3,4]
三角形: 2                   三角形: 3                三角形: 3
```

### 顶点重用

```javascript
// 使用索引缓冲复用顶点
const vertices = [
  0, 1, 0,   // v0: 顶部
  -1, 0, 0,  // v1: 左下
  1, 0, 0,   // v2: 右下
  0, -1, 0   // v3: 底部
];

// 两个三角形共享 v1 和 v2
const indices = [0, 1, 2, 1, 3, 2];

// 绘制
gl.drawElements(gl.TRIANGLES, 6, gl.UNSIGNED_SHORT, 0);
```

## 裁剪

### 视锥体裁剪

在裁剪空间中，可见区域定义为：

```
-w <= x <= w
-w <= y <= w
-w <= z <= w  (WebGL 2.0)
 0 <= z <= w  (WebGL 1.0，可配置)
```

### 裁剪结果

```
┌───────────────────────────────────────┐
│                                       │
│     视锥体                            │
│     ┌─────────────┐                   │
│     │             │                   │
│     │   可见区域   │                   │
│     │             │                   │
│     └─────────────┘                   │
│                                       │
│  A: 完全在内 → 保留                    │
│  B: 完全在外 → 丢弃                    │
│  C: 部分在内 → 裁剪                    │
│                                       │
└───────────────────────────────────────┘

裁剪前:              裁剪后:
  ────────────        ────────────
      ╲                    |
       ╲  三角形     →     |╲
        ╲                  | ╲
         ╲                 新顶点
    视锥体边界
```

### 近平面裁剪

防止物体穿过相机：

```
         eye
          │
          │  near
          │   │
          ├───┼───────────
          │   │
          │   │
          │
          裁剪发生在这里
```

## 透视除法

### 齐次坐标到 NDC

裁剪后执行透视除法：

```
裁剪坐标 (x, y, z, w) → NDC (x/w, y/w, z/w)
```

NDC（标准化设备坐标）范围：
- x: [-1, 1]
- y: [-1, 1]
- z: [-1, 1] 或 [0, 1]

## 背面剔除

### 判断正反面

通过三角形顶点的绕序判断：

```
正面（逆时针）:          背面（顺时针）:
     v0                      v0
    /  \                    /  \
   /    \                  /    \
  v1────v2                v2────v1
  
绕序: v0→v1→v2 (CCW)      绕序: v0→v2→v1 (CW)
```

### 启用背面剔除

```javascript
// 启用背面剔除
gl.enable(gl.CULL_FACE);

// 设置剔除哪一面
gl.cullFace(gl.BACK);   // 剔除背面（默认）
gl.cullFace(gl.FRONT);  // 剔除正面
gl.cullFace(gl.FRONT_AND_BACK);  // 剔除两面

// 设置正面的绕序
gl.frontFace(gl.CCW);   // 逆时针为正面（默认）
gl.frontFace(gl.CW);    // 顺时针为正面
```

### 背面剔除的影响

```
开启背面剔除:            关闭背面剔除:
┌─────────────┐         ┌─────────────┐
│    ┌───┐    │         │    ┌───┐    │
│    │   │    │         │    │   │    │
│    └───┘    │         │    └───┘    │
│             │         │    (背面    │
│             │         │     可见)   │
└─────────────┘         └─────────────┘

性能提升约 50%           需要渲染两面时使用
```

## 光栅化

### 光栅化过程

将图元转换为片元：

```
三角形                  采样点                   片元
   ╲                   ┌───────────┐            ┌───────────┐
    ╲                  │ · · · · · │            │     ███   │
     ╲                 │ · · · · · │     →      │   █████   │
──────╲                │ · · · · · │            │ ███████   │
                       │ · · · · · │            │█████████  │
                       └───────────┘            └───────────┘
                       
每个像素中心判断       被三角形覆盖的像素
是否在三角形内         成为片元
```

### 采样规则

默认在像素中心采样：

```
像素网格:
┌───┬───┬───┬───┐
│ · │ · │ · │ · │  · = 采样点（像素中心）
├───┼───┼───┼───┤
│ · │ · │ · │ · │
├───┼───┼───┼───┤
│ · │ · │ · │ · │
└───┴───┴───┴───┘

采样点在三角形内 → 生成片元
采样点在三角形外 → 不生成
```

### 边缘规则

处理共享边缘的三角形：

```
Top-Left Rule:
┌───────────────────┐
│        ╲          │
│         ╲ 属于    │
│    ──────●────────│  ← 水平边属于左侧三角形
│          ╲        │
│           ╲       │
└───────────────────┘

确保共享边不会被渲染两次
```

## 属性插值

### 重心坐标

用于在三角形内插值属性：

```
           v0
          /│\
         / │ \
        /  │  \
       /   P   \
      /   /│\   \
     /   / │ \   \
    v1 ─┴──┴──┴─ v2
    
P = α·v0 + β·v1 + γ·v2
其中 α + β + γ = 1
```

### 线性插值 vs 透视校正插值

```glsl
// 着色器中的 varying 默认使用透视校正插值
in vec2 v_texCoord;  // 自动透视校正

// 如果需要线性插值（屏幕空间）
flat in int v_id;           // 不插值，使用第一个顶点的值
noperspective in vec2 v_uv; // 屏幕空间线性插值
```

### 插值效果对比

```
透视校正:                屏幕空间线性:
┌─────────────────┐      ┌─────────────────┐
│     ╱╲          │      │     ╱╲          │
│    ╱  ╲         │      │    ╱  ╲         │
│   ╱────╲ 正确   │      │   ╱────╲ 扭曲   │
│  ╱──────╲       │      │  ╱──────╲       │
│ ╱────────╲      │      │ ╱────────╲      │
└─────────────────┘      └─────────────────┘

纹理映射需要透视校正    UI等2D场景可用线性
```

## 视口变换

### NDC 到屏幕坐标

```
NDC:                     屏幕坐标:
    y ↑                       ┌───────────┐
      │                       │           │
  ────┼───→ x               │     y     │
      │                       │     ↑     │
    [-1,1]                    └─────┴─────→ x
                                  (0,0)
```

### 视口设置

```javascript
// 设置视口
gl.viewport(x, y, width, height);

// 全屏
gl.viewport(0, 0, canvas.width, canvas.height);

// 分屏渲染
gl.viewport(0, 0, canvas.width / 2, canvas.height);  // 左半屏
gl.viewport(canvas.width / 2, 0, canvas.width / 2, canvas.height);  // 右半屏
```

### 深度范围

```javascript
// 设置深度范围（NDC z 映射到深度缓冲）
gl.depthRange(near, far);  // 默认 (0.0, 1.0)

// 深度反转（提高深度精度）
gl.depthRange(1.0, 0.0);
gl.clearDepth(0.0);
gl.depthFunc(gl.GREATER);
```

## 多重采样抗锯齿

### MSAA 原理

```
无 MSAA:               4x MSAA:
┌───────────┐         ┌───────────┐
│     ████  │         │    ▓███   │
│   ████████│         │   ▓██████ │
│ ██████████│         │  ▓████████│
│████████   │         │ ▓███████  │
└───────────┘         └───────────┘
 锯齿明显               边缘平滑
 
● = 采样点             每个像素多个采样点
```

### 启用 MSAA

```javascript
// 创建支持 MSAA 的上下文
const gl = canvas.getContext('webgl2', {
  antialias: true  // 启用默认 MSAA
});

// 使用多重采样渲染缓冲（自定义 MSAA）
const rbo = gl.createRenderbuffer();
gl.bindRenderbuffer(gl.RENDERBUFFER, rbo);
gl.renderbufferStorageMultisample(
  gl.RENDERBUFFER,
  4,  // 采样数：4x MSAA
  gl.RGBA8,
  width,
  height
);
```

## 本章小结

- 图元装配将顶点组装成点、线、三角形
- 裁剪移除视锥体外的图元
- 透视除法将裁剪坐标转换为 NDC
- 背面剔除跳过不可见的三角形背面
- 光栅化确定图元覆盖的像素
- 属性使用重心坐标进行透视校正插值
- MSAA 通过多重采样实现抗锯齿
