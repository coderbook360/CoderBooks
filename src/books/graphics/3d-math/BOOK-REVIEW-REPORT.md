## 三、新增核心章节说明

### 3.1 第17章：法线变换与逆转置矩阵

**重要性**: ⭐⭐⭐⭐⭐ （极其重要）

**为什么必须包含**：
- 法线不能直接用模型矩阵变换（会出错）
- 需要使用逆转置矩阵
- 光照计算的核心基础
- 几乎所有3D引擎都需要处理这个问题

**涵盖内容**：
- 为什么法线变换特殊
- 逆转置矩阵的数学推导
- 代码实现
- 在光照中的应用

### 3.2 第18章：Look-At矩阵构建详解

**重要性**: ⭐⭐⭐⭐⭐

**为什么必须包含**：
- 相机系统的核心
- 观察变换的基础
- Three.js 的 camera.lookAt() 的实现原理
- 几乎所有3D应用都需要

**涵盖内容**：
- Look-At矩阵的作用
- 从目标点和上向量构建观察矩阵
- 右手坐标系 vs 左手坐标系
- 代码实现和应用

### 3.3 第23章：切线空间与TBN矩阵

**重要性**: ⭐⭐⭐⭐

**为什么应该包含**：
- 法线贴图的数学基础
- 现代3D渲染必备知识
- PBR材质的基础
- Three.js 自动处理，但理解原理很重要

**涵盖内容**：
- 什么是切线空间
- TBN矩阵的构建（Tangent, Bitangent, Normal）
- 从对象空间到切线空间的转换
- 在着色器中的应用

### 3.4 第28章：深度缓冲与深度精度

**重要性**: ⭐⭐⭐⭐

**为什么包含**：
- 理解Z-fighting的原因
- 深度测试的原理
- 投影矩阵与深度精度的关系
- near/far平面设置的影响

**涵盖内容**：
- 深度缓冲的工作原理
- 深度值的非线性分布
- 深度精度问题
- 如何选择near和far平面

### 3.5 第42章：矩阵分解与重组

**重要性**: ⭐⭐⭐

**为什么有用**：
- 从变换矩阵提取平移、旋转、缩放
- 插值动画需要
- 调试和工具开发
- Three.js 有 decompose 方法

**涵盖内容**：
- 矩阵分解的数学原理
- 提取平移分量
- 提取缩放分量
- 提取旋转（转为四元数）
- 代码实现

### 3.6 第45章：实战案例 - 相机控制器

**重要性**: ⭐⭐⭐⭐

**为什么重要**：
- 综合应用所有知识
- 实际项目必备功能
- 完整的代码示例
- Three.js OrbitControls 的实现原理

**涵盖内容**：
- 轨道相机（Orbit Camera）实现
