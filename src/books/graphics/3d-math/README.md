# 3D 图形学数学基础

本书系统讲解 3D 图形学必备的数学知识，为 WebGL 编程和 Three.js 学习打下坚实的数学基础。

## 📚 本书定位

- **系列位置**：图形学系列 Book 2
- **前置知识**：Book 1 - Canvas 图形编程实战
- **后续学习**：Book 3 - WebGL 基础与渲染管线
- **学习周期**：3-4 周

## 🎯 学习目标

1. 掌握 3D 图形学核心数学工具
2. 理解向量、矩阵、四元数的原理和应用
3. 熟悉 3D 坐标系统和投影变换
4. 能够用 JavaScript 实现完整的 3D 数学库
5. 为学习 WebGL 和 Three.js 做好准备
