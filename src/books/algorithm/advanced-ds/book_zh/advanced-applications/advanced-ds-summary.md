## 数据结构选型速查

### 一维区间问题

| 需求 | 推荐结构 | 复杂度 |
|------|---------|--------|
| 静态区间和 | 前缀和 | O(1) |
| 静态区间最值 | ST 表 | O(1) |
| 单点修改 + 前缀和 | 树状数组 | O(log n) |
| 单点修改 + 区间最值 | 线段树 | O(log n) |
| 区间修改 + 区间查询 | 线段树 + 懒标记 | O(log n) |
| 区间第 K 小（静态） | 主席树 | O(log n) |
| 区间第 K 小（动态） | 树状数组套主席树 | O(log² n) |
| 区间众数 | 分块 | O(√n) |

### 树上问题

| 需求 | 推荐结构 | 复杂度 |
|------|---------|--------|
| LCA（在线） | 倍增 | O(log n) |
| LCA（离线） | Tarjan | O(α(n)) |
| 路径修改/查询 | 树链剖分 | O(log² n) |
| 子树统计 | DFS 序 + BIT | O(log n) |
| 子树颜色统计 | DSU on Tree | O(n log n) |
| 稀疏关键点 | 虚树 | O(k log k) |

### 字符串问题

| 需求 | 推荐结构 | 复杂度 |
|------|---------|--------|
| 前缀查询 | Trie | O(m) |
| 多模式匹配 | AC 自动机 | O(n + m) |
| 后缀排序 | 后缀数组 | O(n log n) |
| 子串统计 | 后缀自动机 | O(n) |
| 最长回文 | Manacher | O(n) |

### 多维问题

| 需求 | 推荐结构 | 复杂度 |
|------|---------|--------|
| 二维区间和（静态） | 二维前缀和 | O(1) |
| 二维区间和（动态） | 二维树状数组 | O(log² n) |
| 矩形面积并 | 扫描线 | O(n log n) |
| 三维偏序 | CDQ 分治 | O(n log² n) |
| 多维最近邻 | K-D 树 | O(log n) ~ O(n) |

## 核心技术总结

### 1. 分治思想

- **归并排序**：逆序对、区间计数
- **线段树**：区间分解
- **CDQ 分治**：降维处理偏序
- **树分治**：点分治、边分治

### 2. 懒惰求值

- **懒标记**：延迟更新
- **可持久化**：按需复制
- **动态开点**：按需分配

### 3. 离散化与压缩

- **值域压缩**：大值域 → 小值域
- **坐标压缩**：扫描线优化
- **虚树**：大树 → 小树

### 4. 启发式策略

- **按秩合并**：并查集优化
- **重链优先**：树链剖分
- **小合并到大**：DSU on Tree

## 结语

高级数据结构是算法竞赛和工程实践的重要工具。掌握这些结构，不仅能解决复杂问题，更能培养抽象思维和工程能力。

**学习建议**：
1. **理解原理**：不要死记模板，理解设计思想
2. **动手实现**：自己从零实现每个数据结构
3. **刷题巩固**：通过题目加深理解
4. **总结归纳**：建立自己的知识体系

**进阶方向**：
- 函数式数据结构
- 并发数据结构
- 外存数据结构
- 近似算法与概率数据结构
