# LeetCode 算法通关之路（五）：高级数据结构篇

掌握解决复杂问题的终极武器——高级数据结构。

## 🎯 你将学到什么

本书系统介绍算法竞赛和工程实践中最重要的高级数据结构：

- **字典树 Trie** - 高效字符串检索与前缀匹配
- **线段树** - 区间查询与修改的通用解决方案
- **树状数组** - 简洁高效的前缀和维护
- **平衡树** - 有序集合的动态维护
- **分块与莫队** - 根号复杂度的离线算法
- **后缀结构** - 后缀数组与后缀自动机
- **可持久化** - 历史版本查询与函数式数据结构
- **树链剖分** - 树上路径问题的高效解法
- **高级应用** - CDQ分治、整体二分、K-D树
