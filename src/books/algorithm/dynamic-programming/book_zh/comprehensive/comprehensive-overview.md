# 综合实战概述

本章将综合运用前面学到的所有 DP 技巧，解决复杂的综合性问题。

## 为什么需要综合实战

前面各章学习了：

- **基础 DP**：状态定义、转移方程
- **记忆化搜索**：递归 + 缓存
- **线性 DP**：序列上的 DP
- **背包问题**：选择与组合
- **状态机 DP**：状态转移建模
- **序列 DP**：子序列问题
- **区间 DP**：区间合并/分割
- **博弈 DP**：对抗决策
- **状态压缩 DP**：集合状态
- **数位 DP**：数字位处理
- **树形 DP**：树结构 DP
- **DP 优化**：时空优化技巧

真实问题往往需要综合运用多种技巧。

## 综合题目的特点

### 1. 多种技巧组合

一道题可能同时用到：
- 状态压缩 + 区间 DP
- 树形 DP + 背包
- 博弈 + 记忆化
- 数位 DP + 前缀和

### 2. 复杂状态设计

需要从问题中抽象出合适的状态：
- 多维状态
- 复合状态
- 压缩状态

### 3. 多步推导

从问题到解法需要多步推导：
1. 理解问题
2. 抽象模型
3. 设计状态
4. 推导转移
5. 优化实现

## 解题方法论

### Step 1：理解问题

- 输入是什么？
- 输出是什么？
- 有什么约束？
- 规模有多大？

### Step 2：识别 DP 类型

- 有无明显的阶段划分？
- 是否存在最优子结构？
- 子问题是否重叠？

### Step 3：设计状态

- 需要记录什么信息？
- 信息如何组织？
- 状态空间有多大？

### Step 4：推导转移

- 状态如何变化？
- 决策有哪些？
- 如何组合子问题答案？

### Step 5：实现与优化

- 递归还是迭代？
- 空间能否优化？
- 时间能否优化？
