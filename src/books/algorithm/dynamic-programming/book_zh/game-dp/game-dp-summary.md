# 博弈 DP 总结

## 博弈问题分类

### 1. 必胜必败型

判断先手是否必胜。

**特点**：
- 状态有限
- 结果只有胜/败两种
- 可以找到数学规律

**典型题目**：
- Nim 游戏
- 除数博弈

**核心思路**：
```
必胜态 = 存在一步让对手进入必败态
必败态 = 所有选择都让对手进入必胜态
```

### 2. 最优收益型

双方都采取最优策略时，求先手的最大收益。

**特点**：
- 通常有分数
- 用区间 DP
- 相对收益表示法

**典型题目**：
- 石子游戏
- 预测赢家

**核心公式**：
```
dp[i][j] = max(选择 k - dp[下一状态])
```

### 3. 复杂博弈

涉及多维状态的博弈。

**特点**：
- 状态复杂
- 可能有平局
- 需要反向 BFS

**典型题目**：
- 猫和老鼠
- 石子游戏 II/III

## 常用技巧总结

### 技巧一：相对收益

不用区分 Max/Min 玩家，用相对收益简化：

```typescript
// dp[state] = 当前玩家比对手多获得的分数
dp[state] = max(score[action] - dp[next_state])
```

### 技巧二：后缀和预处理

快速计算区间和：

```typescript
// 后缀和
const suffix = new Array(n + 1).fill(0);
for (let i = n - 1; i >= 0; i--) {
  suffix[i] = suffix[i + 1] + values[i];
}

// dp[i] = suffix[i] - dp[next]
```

### 技巧三：奇偶性分析

很多博弈问题有奇偶规律：

- 单堆 Nim：`n % 4 !== 0` 必胜
- 除数博弈：偶数必胜
- 多堆 Nim：XOR ≠ 0 必胜

### 技巧四：区间 DP 模板

```typescript
function gameDP(values: number[]): number {
  const n = values.length;
  const dp = Array.from({ length: n }, () => new Array(n).fill(0));
  
  // base case
  for (let i = 0; i < n; i++) {
    dp[i][i] = values[i];
  }
  
  // 枚举区间长度
  for (let len = 2; len <= n; len++) {
    for (let i = 0; i + len - 1 < n; i++) {
      const j = i + len - 1;
      // 选左边 vs 选右边
      dp[i][j] = Math.max(
        values[i] - dp[i + 1][j],
        values[j] - dp[i][j - 1]
      );
    }
  }
  
  return dp[0][n - 1];
}
```

### 技巧五：反向 BFS

适用于图上博弈：

// 1. 确定终止状态
// 2. 从终止状态反向传播

// 4. 剩余状态为平局

## 调试技巧

1. **打表找规律**：先用 DP 算出小规模结果，观察规律
2. **边界检查**：注意 `n = 1`、空数组等边界
3. **验证对称性**：很多博弈问题有对称结构

## 面试建议

1. **先用 DP**：即使有数学规律，面试时也建议先写 DP
2. **解释思路**：清楚解释极大极小思想
3. **简化表示**：用相对收益而非绝对收益
4. **提及优化**：知道有数学规律是加分项

## 经典公式速查

| 问题类型 | 核心公式 |
|---------|----------|
| 必胜必败 | `win[s] = any(!win[next])` |
| 两端取数 | `dp[i][j] = max(v[i] - dp[i+1][j], v[j] - dp[i][j-1])` |
| 开头取数 | `dp[i] = max(sum[i:i+k] - dp[i+k])` |
| 多堆 Nim | `xor(all piles) !== 0` |

## 本章小结

博弈 DP 的核心是**站在当前玩家的角度**思考：

1. 我要最大化我的收益
2. 对手也会最大化他的收益
3. 用"我的收益 - 对手最优收益"简化思考

掌握了这个对抗性思维，大部分博弈问题都能迎刃而解。
