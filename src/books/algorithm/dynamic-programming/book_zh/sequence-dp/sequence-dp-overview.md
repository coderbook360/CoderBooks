# 序列 DP 概述

序列 DP 是最常见的 DP 类型之一，处理的是一个或多个序列上的最优化问题。

## 什么是序列 DP

**序列 DP** 的特点：
- 输入是一个或多个序列（数组、字符串）
- 状态通常与序列的位置/索引相关
- 答案是某种最优解（最长、最短、计数等）

## 序列 DP 的分类

### 单序列问题

只涉及一个序列：

| 类型 | 典型问题 | 状态定义 |
|-----|---------|---------|
| 最长递增子序列 | LIS | `dp[i]` = 以 i 结尾的最长长度 |
| 最大子数组 | Kadane | `dp[i]` = 以 i 结尾的最大和 |
| 解码方法 | 91 | `dp[i]` = 前 i 个字符的方案数 |

### 双序列问题

涉及两个序列：

| 类型 | 典型问题 | 状态定义 |
|-----|---------|---------|
| 最长公共子序列 | LCS | `dp[i][j]` = s1[0..i] 和 s2[0..j] 的 LCS |
| 编辑距离 | Edit Distance | `dp[i][j]` = s1[0..i] 变成 s2[0..j] 的最小操作 |
| 交错字符串 | Interleaving | `dp[i][j]` = s1[0..i] 和 s2[0..j] 能否交错成 s3 |

## 核心模式

### 模式一：以位置 i 结尾

```
dp[i] = 以 nums[i] 结尾的最优解
```

**适用场景**：
- 最长递增子序列
- 最大子数组和
- 最大子数组乘积

**转移方式**：
```typescript
for (let i = 0; i < n; i++) {
  for (let j = 0; j < i; j++) {
    if (满足条件) {
      dp[i] = optimize(dp[i], dp[j] + 贡献);
    }
  }
}
```

### 模式二：前 i 个元素

```
dp[i] = 前 i 个元素的最优解
```

**适用场景**：
- 单词拆分
- 解码方法
- 打家劫舍

**转移方式**：
```typescript
for (let i = 1; i <= n; i++) {
  for (let j = 0; j < i; j++) {
    if (满足条件) {
      dp[i] = optimize(dp[i], dp[j] + 贡献);
    }
  }
}
```

### 模式三：双序列匹配

```
dp[i][j] = s1[0..i] 和 s2[0..j] 的最优解
```

**适用场景**：
- 最长公共子序列
- 编辑距离
- 正则表达式匹配

**转移方式**：
```typescript
for (let i = 1; i <= m; i++) {
  for (let j = 1; j <= n; j++) {
    if (s1[i-1] === s2[j-1]) {
      dp[i][j] = dp[i-1][j-1] + 1;
    } else {
      dp[i][j] = optimize(dp[i-1][j], dp[i][j-1], dp[i-1][j-1]);
    }
  }
}
```

## 两种状态定义的对比

### 以 i 结尾 vs 前 i 个

```typescript
// 以 i 结尾（必须包含 nums[i]）
dp[i] = 以 nums[i] 结尾的最长递增子序列长度

// 前 i 个（不一定包含 nums[i]）
dp[i] = 前 i 个元素中的最长递增子序列长度
```

**区别**：
- "以 i 结尾"：答案是 `max(dp[0], dp[1], ..., dp[n-1])`
- "前 i 个"：答案是 `dp[n]`

**选择建议**：
- 如果子问题之间需要"连接"（如 LIS），用"以 i 结尾"
- 如果子问题是"累积"的（如单词拆分），用"前 i 个"

## 子序列 vs 子数组

| 概念 | 定义 | 示例 |
|-----|------|------|
| 子序列 | 不连续，但保持相对顺序 | [1,3,5] 是 [1,2,3,4,5] 的子序列 |
| 子数组 | 连续的元素 | [2,3,4] 是 [1,2,3,4,5] 的子数组 |

**对应问题**：
- 子序列：LIS、LCS
- 子数组：最大子数组和、最长重复子数组

## 本章将学习的问题

### 单序列

- **最长递增子序列（LIS）**：经典的 O(n²) 和 O(n log n) 解法
- **最长递增子序列的个数**：LIS 的变种
- **最大子数组乘积**：处理负数的技巧
- **删除与获得点数**：打家劫舍变种

### 双序列

- **最长公共子序列（LCS）**：双序列的基础
- **编辑距离**：字符串相似度的经典问题
- **不同的子序列**：计数类问题
- **最长重复子数组**：子数组版本的 LCS
- **交错字符串**：三序列问题

### 其他

- **最长字符串链**：LIS 的变种
- **俄罗斯套娃信封**：二维 LIS

## 解题模板

### 单序列模板

```typescript
function solveSingleSequence(nums: number[]): number {
  const n = nums.length;
  const dp = new Array(n).fill(初始值);
  
  for (let i = 0; i < n; i++) {
    for (let j = 0; j < i; j++) {
      if (满足条件(nums[i], nums[j])) {
        dp[i] = optimize(dp[i], dp[j] + 贡献);
      }
    }
  }
  
  return aggregate(dp);  // max, min, sum 等
}
```

### 双序列模板

```typescript
function solveTwoSequences(s1: string, s2: string): number {
  const m = s1.length, n = s2.length;
  const dp: number[][] = Array.from(
    { length: m + 1 },
    () => new Array(n + 1).fill(初始值)
  );
  
  // 边界初始化
  for (let i = 0; i <= m; i++) dp[i][0] = 边界值;
  for (let j = 0; j <= n; j++) dp[0][j] = 边界值;
  
  for (let i = 1; i <= m; i++) {
    for (let j = 1; j <= n; j++) {
      if (s1[i-1] === s2[j-1]) {
        dp[i][j] = dp[i-1][j-1] + 1;
      } else {
        dp[i][j] = optimize(dp[i-1][j], dp[i][j-1]);
      }
    }
  }
  
  return dp[m][n];
}
```

## 本章小结

1. **序列 DP 特点**：状态与位置相关
2. **单序列**：以 i 结尾或前 i 个
3. **双序列**：dp[i][j] 表示两个序列的前缀
4. **核心技巧**：正确的状态定义和转移方程
