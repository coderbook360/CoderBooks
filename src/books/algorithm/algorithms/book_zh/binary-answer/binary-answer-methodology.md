# 二分答案方法论

二分答案是二分查找的高级应用。我们不再是在数组中查找元素，而是在**答案空间**中查找满足条件的最优解。

---

## 什么是二分答案？

假设题目要求找一个最小的 x，使得满足某个条件 F(x)。

如果 F(x) 具有**单调性**：
- 存在某个分界点 x*
- x < x* 时，F(x) = false
- x >= x* 时，F(x) = true

那么我们可以二分这个 x*。

---

## 典型场景

### 1. 最小化最大值

例如：将数组分成 k 段，使得每段和的**最大值最小**。

```
答案空间：[max(nums), sum(nums)]
         最小可能    最大可能

二分一个答案 mid，判断能否在"每段和不超过 mid"的约束下分成 k 段
```

### 2. 最大化最小值

例如：在 n 个位置放 k 个球，使得相邻球的**最小距离最大**。

```
答案空间：[1, maxPosition - minPosition]

二分一个答案 mid，判断能否在"相邻距离至少为 mid"的约束下放 k 个球
```

### 3. 求满足条件的边界值

例如：吃香蕉的最小速度、运货的最小运力。

      right = mid;
    } else {
      left = mid + 1;
    }
  }
  
  return left;
}

## 二分答案 vs 普通二分

| 特点 | 普通二分 | 二分答案 |
|-----|---------|---------|
| 搜索对象 | 数组元素 | 答案空间 |
| 判断方式 | 比较大小 | check 函数 |
| 难点 | 边界处理 | 设计 check 函数 |

---

## 识别二分答案的信号

1. 题目要求"最小的最大值"或"最大的最小值"
2. 答案有明确的范围
3. 给定一个答案，可以快速验证是否可行
4. 可行性具有单调性
