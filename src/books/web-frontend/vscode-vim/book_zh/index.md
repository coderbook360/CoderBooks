# VSCode Vim 10x 效率指南：完全键盘驱动的开发工作流

本书专为追求极致效率的开发者打造，聚焦 VSCode Vim 实战技巧，帮助你快速建立全键盘工作流，实现 10 倍效率提升。

## 🎯 你将学到什么

- **VSCode Vim 完整配置** - 从安装到高级定制的完整指南
- **Vim 核心操作** - 操作符、文本对象、重复等核心概念
- **文件与项目管理** - 全键盘的文件导航和项目切换
- **代码导航技巧** - Symbol 跳转、Peek 定义等高效导航方法
- **多光标与批量编辑** - VSCode 与 Vim 结合的强大编辑能力
- **前端开发场景** - React、TypeScript、CSS 等实战技巧

## 📚 本书结构

本书共分为 **13 个部分，51 个章节**：

1. **快速上手与环境配置** - VSCode Vim 安装、配置和优化
2. **Vim 核心操作精通** - 掌握 Vim 的核心编辑思维
3. **文件与项目管理** - 全键盘的文件操作
4. **代码导航与定位** - 高效的代码跳转和查找
5. **快速移动技巧** - EasyMotion、Sneak 等插件使用
6. **VSCode 集成功能** - 命令面板、IntelliSense、快速修复
7. **多光标与批量编辑** - 批量重构和代码生成
8. **窗口与侧边栏管理** - 全键盘的窗口布局控制
9. **Git 集成操作** - 全键盘的版本控制工作流
10. **前端开发场景** - React、TypeScript、CSS 实战
11. **诊断与代码质量** - 错误处理、代码格式化
12. **高级技巧与优化** - 自定义快捷键、宏录制等
