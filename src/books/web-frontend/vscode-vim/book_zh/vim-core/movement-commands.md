# 移动命令完全指南

在 Vim 中，光标移动不是用鼠标点击，而是用"命令导航"。掌握移动命令，就像掌握了代码的"传送门"——你可以在毫秒间到达任何位置。

本章将系统讲解所有移动命令，从基础到高级，帮你建立完整的导航能力。

## 基础移动：hjkl

放弃方向键，使用 `hjkl` 移动光标：

```
     k
     ↑
h ←     → l
     ↓
     j
```

- `h`：左移一个字符
- `j`：下移一行
- `k`：上移一行
- `l`：右移一个字符

为什么不用方向键？因为方向键在键盘右下角，手指需要离开主键区。`hjkl` 都在主键区，右手食指自然落在 `j` 上。

### 数字前缀

所有移动命令都可以加数字前缀：

| 命令 | 效果 |
|------|------|
| `5j` | 向下移动 5 行 |
| `10k` | 向上移动 10 行 |
| `3l` | 向右移动 3 个字符 |

这比连续按 `jjjjj` 快得多。

### 适应期

刚开始用 `hjkl` 会很不习惯。建议：

1. 把箭头键想象成"不存在"
2. 每次想用箭头键时，停下来用 `hjkl`
3. 坚持一周，形成肌肉记忆

一旦适应，你会发现手完全不需要离开主键区。

## 词级移动

字符级移动太慢了。词级移动是日常编辑的主力：

| 命令 | 效果 | 助记 |
|------|------|------|
| `w` | 下一个词首 | word |
| `b` | 上一个词首 | back |
| `e` | 当前词尾 | end |
| `W` | 下一个 WORD 首 | WORD |
| `B` | 上一个 WORD 首 | Back |
| `E` | 当前 WORD 尾 | End |

### word vs WORD

小写的 `w/b/e` 按"词"移动，词的边界是：字母、数字、下划线之外的任何字符。

大写的 `W/B/E` 按"WORD"移动，WORD 的边界只有空格。

看一个例子：

```javascript
const user_name = "John Doe";
```

从行首用 `w` 移动：
1. `w` → 到 `user_name`
2. `w` → 到 `=`
3. `w` → 到 `"John`
4. `w` → 到 `Doe"`

用 `W` 移动：
1. `W` → 到 `user_name`
2. `W` → 到 `=`
3. `W` → 到 `"John`

`W` 跳过更多，适合快速浏览。`w` 更精确，适合精细编辑。

### 组合数字

| 命令 | 效果 |
|------|------|
| `3w` | 前进 3 个词 |
| `5b` | 后退 5 个词 |

## 行内移动

行内有很多精确定位的方法：

| 命令 | 效果 | 助记 |
|------|------|------|
| `0` | 行首（第 0 列） | 零 |
| `^` | 第一个非空字符 | regex 行首 |
| `$` | 行尾 | regex 行尾 |
| `g_` | 最后一个非空字符 | |

### 0 vs ^

`0` 是绝对行首（第 0 列），`^` 是第一个非空字符。

```javascript
    const x = 1;  // 假设有 4 个空格缩进
```

- `0` 定位到第一个空格
- `^` 定位到 `c`

大多数时候你想要 `^`，因为缩进通常不是你的目标。

可以映射 `H` 和 `L` 为 `^` 和 `$`，更好按：

```json
{
  "vim.normalModeKeyBindingsNonRecursive": [
    { "before": ["H"], "after": ["^"] },
    { "before": ["L"], "after": ["$"] }
  ]
}
```

### f/t 精确跳转

`f` 和 `t` 是行内精确跳转的利器：

| 命令 | 效果 | 助记 |
|------|------|------|
| `f{char}` | 跳到字符 | find |
| `F{char}` | 反向跳到字符 | Find |
| `t{char}` | 跳到字符前 | till |
| `T{char}` | 反向跳到字符前 | Till |
| `;` | 重复上次 f/t | |
| `,` | 反向重复 | |

**实战示例**：

```javascript
function getUserProfile(userId) {
```

光标在行首，想到达 `userId`：

1. `fu` → 跳到第一个 `u`（在 function 里）
2. `;` → 跳到下一个 `u`（在 User 里）
3. `;` → 跳到再下一个 `u`（在 userId 里）

或者直接：`2fu` → 跳到第二个 `u`

### f vs t 的选择

`f` 停在目标字符上，`t` 停在目标字符前。

什么时候用 `t`？当你要配合操作符，不想包含目标字符时：

```javascript
const name = "John";
```

- `dt"` → 删除到引号前，保留引号
- `df"` → 删除到引号，包含引号

## 跨行移动

### 行跳转

| 命令 | 效果 |
|------|------|
| `gg` | 跳到文件第一行 |
| `G` | 跳到文件最后一行 |
| `{n}G` | 跳到第 n 行 |
| `{n}gg` | 跳到第 n 行（同上） |
| `:{n}` | 跳到第 n 行（命令方式） |

跳转到具体行号常用于：
- 错误信息提示"第 42 行有错误"→ `42G`
- 代码审查评论"请看第 100 行"→ `100G`

### 翻页

| 命令 | 效果 | 助记 |
|------|------|------|
| `Ctrl+D` | 向下半页 | Down |
| `Ctrl+U` | 向上半页 | Up |
| `Ctrl+F` | 向下整页 | Forward |
| `Ctrl+B` | 向上整页 | Backward |

半页翻动（`Ctrl+D/U`）是最常用的，因为：
1. 保持上下文——翻动后仍能看到之前的部分内容
2. 比整页更容易追踪位置

### 段落移动

| 命令 | 效果 |
|------|------|
| `{` | 上一个空行（段落首） |
| `}` | 下一个空行（段落尾） |

在代码中，段落通常是函数或代码块。`{` 和 `}` 可以快速在函数间跳转。

## 屏幕相对移动

有时你想跳转到屏幕上的某个位置，而不是文件的某个位置：

| 命令 | 效果 | 助记 |
|------|------|------|
| `H` | 屏幕顶部 | High |
| `M` | 屏幕中部 | Middle |
| `L` | 屏幕底部 | Low |

这三个命令很适合"先大概到那块区域，再精细定位"。

### 滚动调整

| 命令 | 效果 |
|------|------|
| `zz` | 当前行居中 |
| `zt` | 当前行置顶 |
| `zb` | 当前行置底 |

`zz` 特别有用——当你跳转到某行后，用 `zz` 让它居中，方便查看上下文。

## 搜索移动

搜索不只是查找，更是快速移动的工具。

| 命令 | 效果 |
|------|------|
| `/pattern` | 向下搜索 |
| `?pattern` | 向上搜索 |
| `n` | 下一个匹配 |
| `N` | 上一个匹配 |
| `*` | 搜索当前词（向下） |
| `#` | 搜索当前词（向上） |

### * 和 # 的威力

`*` 和 `#` 是被低估的移动命令。

光标在 `handleClick` 上，按 `*`：
1. 自动用 `handleClick` 作为搜索词
2. 跳到下一个 `handleClick`

这比手动输入 `/handleClick` 快得多。`#` 则反向搜索。

## 标记系统

标记（mark）让你可以"记住"位置，随时跳回：

| 命令 | 效果 |
|------|------|
| `m{a-z}` | 设置标记 |
| `` `{a-z} `` | 跳转到标记（精确位置） |
| `'{a-z}` | 跳转到标记（行首） |

小写标记（a-z）是文件内的，大写标记（A-Z）是跨文件的。

**实战场景**：

你在编辑文件顶部的某个函数，需要临时跳到文件底部查看另一个函数：

1. `ma` → 设置标记 a
2. `G` → 跳到文件底部
3. 查看代码...
4. `` `a `` → 跳回原位置

## 移动命令选择策略

面对不同距离，选择最优的移动方式：

| 距离 | 推荐方式 |
|------|----------|
| 1-3 个字符 | `h/l` 或 `f/t` |
| 几个词 | `w/b/e` |
| 行内远处 | `f/t` + `;` |
| 行首/行尾 | `^/$` 或 `H/L` 映射 |
| 几行内 | `j/k` 配合数字 |
| 5-20 行 | `Ctrl+D/U` |
| 段落间 | `{/}` |
| 已知行号 | `{n}G` |
| 屏幕内 | `H/M/L` + 微调 |
| 搜索目标 | `/` 或 `*/#` |
| 跨文件的固定位置 | 大写标记 |

## 练习

完成以下练习，巩固移动命令：

1. 打开一个 100+ 行的代码文件
2. 用 `gg` 跳到文件首，用 `G` 跳到文件尾
3. 用 `50G` 跳到第 50 行
4. 用 `zz` 让当前行居中
5. 用 `{` 和 `}` 在函数间跳转
6. 找一个变量名，用 `*` 跳转到下一个使用位置
7. 用 `ma` 设置标记，移动到别处，用 `` `a `` 跳回
8. 在一行中用 `f` 跳到某个字符，用 `;` 继续跳到下一个
