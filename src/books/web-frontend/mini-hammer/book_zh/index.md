# mini-hammer.js 实战：从零构建自己的手势库

本书将带你从零开始，一步步构建一个属于自己的手势库。我们将深入 Hammer.js 的设计精髓，不仅实现核心功能，更将领悟其背后的架构思想与手势识别原理。

## 🎯 你将学到什么

- **手势识别原理** - 状态机与手势生命周期
- **事件系统设计** - EventEmitter 与输入适配
- **核心手势实现** - Tap、Pan、Swipe、Press
- **高级手势与协同** - Pinch、Rotate、手势冲突处理

## 📖 开始阅读

👉 从 [序言](./preface) 开始，或者使用左侧导航菜单浏览各个章节。

## 📚 本书结构

本书共分为 **5 个部分，18 个章节**：

1. **启程** - 手势库概述与 Hammer.js 入门
2. **奠基** - EventEmitter、输入适配、状态机
3. **生长** - Tap、Pan、Swipe、Press 实现
4. **飞跃** - Pinch & Rotate、手势协同
5. **封装** - 最终组装与展望
