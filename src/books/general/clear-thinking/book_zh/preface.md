# 清晰思考：高手思维操作系统

> 一套融合逻辑学原理的可执行思维方法论

---

## 这本书是给谁的

如果你经常遇到这些情况：

- 遇到问题不知道从哪里下手
- 明明想清楚了，但说不清楚
- 做决定总是犹豫，或者凭感觉
- 做完一件事，不知道哪里做得好、哪里可以改进

那这本书可能对你有帮助。

## 这本书讲什么

这本书提供一套系统的思考方法：**CPERF**。

```
C - Capture 捕获：识别问题、收集信息
P - Process 处理：分类问题、结构化分析、做出决策
E - Express 表达：组织输出、清晰表达
R - Review 复盘：检验结果、验证假设
F - Form 固化：形成习惯、持续改进
```

这套方法融合了逻辑学的核心原理：
- 命题与论证
- 演绎与归纳
- 前提与结论
- 常见谬误的识别

但我们不会讲枯燥的理论，而是直接告诉你：
**遇到问题怎么想、怎么做。**

## 这本书怎么用

**第一部分（第1-10章）**是方法论。
系统讲解CPERF的每个环节，包括原理、方法和检查清单。

**第二部分（第11-20章）**是实战应用。
10个真实场景，每个场景展示如何用CPERF处理。

建议的阅读方式：
1. 先快速浏览一遍，了解全貌
2. 遇到实际问题时，翻到对应章节参考
3. 刻意练习，把方法用到工作和生活中

## 期望的结果

读完这本书，你应该能够：

1. **遇到问题知道怎么分析**
   - 不是凭感觉，而是有章法
   
2. **做决定更有依据**
   - 不是拍脑袋，而是有逻辑
   
3. **表达更清晰**
   - 不是东说西说，而是有结构
   
4. **持续改进**
   - 不是做完就完，而是不断变好

## 关于清晰思考

清晰思考不是天赋，是技能。

技能可以通过练习获得。

刚开始用这套方法会觉得慢、费劲，
但用多了会变成习惯，变成直觉。

当清晰思考变成你的思维方式，
你处理问题的效率和质量都会提升。

这就是这本书想帮你做到的。
