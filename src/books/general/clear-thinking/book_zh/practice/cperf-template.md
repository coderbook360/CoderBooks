## 深度版（重大决策）

### 第一阶段：充分捕获（1-2天）

**任务清单**：
- [ ] 明确问题的背景和重要性
- [ ] 列出所有利益相关者
- [ ] 收集各方面信息
- [ ] 验证信息可靠性
- [ ] 识别信息盲点

### 第二阶段：深度加工（2-3天）

**任务清单**：
- [ ] 多角度分析问题
- [ ] 咨询不同专家
- [ ] 使用多个分析框架
- [ ] 评估长期影响
- [ ] 考虑二阶效应

### 第三阶段：组织表达（1天）

**任务清单**：
- [ ] 形成清晰结论
- [ ] 准备完整论证
- [ ] 预判所有反对意见
- [ ] 准备备选方案
- [ ] 设计沟通策略

### 第四阶段：深度复盘（持续）

**任务清单**：
- [ ] 设定关键里程碑
- [ ] 定期检查进展
- [ ] 及时调整方向
- [ ] 完成后全面复盘
- [ ] 提取可复用经验

### 第五阶段：系统固化

**任务清单**：
- [ ] 总结方法论收获
- [ ] 更新个人知识库
- [ ] 分享给团队
- [ ] 形成标准流程
- [ ] 持续迭代优化

## 模板使用指南

### 如何选择版本

| 决策类型 | 推荐版本 | 时间预算 |
|---------|---------|---------|
| 日常小决策 | 快速版 | 5分钟 |
| 一般工作决策 | 标准版 | 30分钟 |
| 重要人生决策 | 深度版 | 数天 |
| 复杂项目决策 | 深度版 | 1-2周 |

### 使用建议

1. **不要跳过步骤**：每个步骤都有价值
2. **书写比思考更清晰**：一定要写下来
3. **迭代使用**：可以多次循环CPERF
4. **保留记录**：方便未来复盘
5. **灵活调整**：根据实际需要增减内容
