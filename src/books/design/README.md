# 前端架构师成长系列

帮助前端开发者系统性地成长为架构师的完整知识体系。

## 系列概述

本系列共 10 本书，涵盖从代码设计到系统架构、从技术深度到领导力的完整知识体系。系列总计约 **780+ 章节**，理论与实战结合，深度解析每个主题。

**目标读者**：3-5 年经验的中高级前端开发者

**核心目标**：帮助读者建立完整的架构知识体系，能够自信地进行架构设计和技术决策

```

       ↓
第二本：工程化架构设计（掌握工程化体系）
```

- **想提升代码质量与设计能力** → 从第一本开始

- **正在构建组件库/设计系统** → 直接阅读第三本
- **准备晋升架构师/技术负责人** → 重点阅读第四本
