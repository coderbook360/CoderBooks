# 序言

## 关于本书

这是《LeetCode 算法通关之路》系列的第二册——**算法技巧篇**。

如果说第一册让你掌握了"工具"（数据结构），那么本书将教你如何"使用工具"（算法技巧）。

## 前置要求

学习本书前，你需要具备：
- ✅ 熟悉数组、链表、栈、队列等基本数据结构
- ✅ 了解哈希表和二叉树的基本操作
- ✅ 掌握递归的基本概念
- ✅ 能够分析算法的时间和空间复杂度

如果以上内容不熟悉，建议先学习本系列第一册《数据结构与编程基础》。

## 本书内容

本书系统讲解算法面试中最常用的解题技巧：

### 核心章节

| 章节 | 内容 | 适用场景 |
|-----|------|---------|
| 递归思维训练 | 递归设计、三要素、递归转迭代 | 所有递归类问题的基础 |
| 双指针技巧 | 对撞指针、快慢指针 | 数组/链表的区间问题 |
| 滑动窗口 | 固定窗口、可变窗口 | 连续子数组/子串问题 |
| 前缀和与差分 | 区间求和、区间修改 | 区间统计类问题 |
| 二分查找 | 三种模板、二分答案 | 有序数据的搜索问题 |
| 贪心算法 | 贪心选择、最优子结构 | 局部最优即全局最优的问题 |
| 分治算法 | 分解、解决、合并 | 可递归分解的问题 |
| 排序算法 | 十大经典排序 | 排序及其变体问题 |
| 回溯算法 | 回溯模板、剪枝优化 | 组合、排列、子集问题 |

### 学习收获

完成本书学习后，你将能够：
- 看到问题能快速识别适用的算法技巧
- 熟练运用双指针、滑动窗口等高频技巧
- 掌握贪心和回溯的思维方式
- 具备解决 LeetCode 中等难度题目的能力

### 与相邻册次的关系

- 本书的递归章节是第一册树遍历的深化

## 学习建议

1. **递归先行**：本书首章是"递归思维训练"，这是后续所有内容的基础
2. **模板记忆**：滑动窗口、二分查找、回溯都有经典模板，要熟练掌握
3. **举一反三**：每种技巧学完后，尝试用它解决更多类似问题
4. **对比学习**：贪心 vs 动态规划，理解何时用贪心、何时需要 DP
