# 关于码书

## 我们是谁

码书（CoderBooks）是一个专注于程序员进阶技术的开源书籍项目。我们致力于用最简单、最易懂的方式，帮助程序员学习和掌握各类技术知识。

## 我们的理念

- **简单易懂**：将复杂的技术概念用通俗易懂的语言解释清楚
- **即学即用**：每本书都聚焦实战，让你学完就能应用到工作中
- **迷你精炼**：每本书专注于一个具体知识点，篇幅适中，不浪费你的时间

## 书籍系列

### Mini 实战系列

我们的 Mini 实战系列专注于从零构建各种工具和库，通过亲手实现来深入理解原理：

- **Mini Acorn**：从零构建 JavaScript 解析器
- **Mini Ramda**：深入 JavaScript 函数式编程
- **Mini Hammer.js**：从零构建手势库
- **Mini Vite**：拆解 Vite 核心运行机制
- **Mini Path-to-RegExp**：理解路由匹配原理

### 方法论系列

- **AI 提示词工程实战**：从新手到架构师的完整学习路径
