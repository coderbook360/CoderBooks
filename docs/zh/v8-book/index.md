# V8 引擎深度解析

> 理解 JavaScript 底层运行机制

本书将带你深入 V8 引擎，从前端开发者的视角理解 JavaScript 代码的底层运行机制。我们不是为了研究 V8 源码而学习，而是通过 V8 的实现来解答你在日常开发中的困惑：为什么这样写代码更快？闭包是如何存储的？异步任务是如何调度的？

## 📖 目录

### 第一部分：V8 架构与执行流程
从源码到机器码的完整旅程，理解解析、编译和优化的全过程。

### 第二部分：基本类型的底层实现
Tagged Pointer、Smi、隐藏类等核心概念，理解 JavaScript 值的真实存储方式。

### 第三部分：高级类型与数据结构
Map、Set、BigInt、TypedArray 等现代 JavaScript 特性的底层实现。

### 第四部分：模块系统与作用域
ESM 模块加载机制、循环依赖处理、作用域链的底层实现。

### 第五部分：执行上下文与闭包
执行上下文、闭包、this 绑定、词法环境等核心概念的深度解析。

### 第六部分：内存管理与垃圾回收
V8 的堆结构、GC 算法、内存泄漏分析与优化策略。

### 第七部分：性能优化
内联缓存、TurboFan 编译器、去优化机制，编写 V8 友好的代码。

### 第八部分：异步机制
事件循环、Promise 内部机制、async/await 转换原理。

### 第九部分：调试机制
错误处理、Source Map、DevTools 调试协议。

### 第十部分：高级主题
原型链、Proxy、Symbol、正则引擎、WebAssembly 等高级主题。
