# 实战：划分字母区间

这道题展示了贪心在字符串分割中的应用。

## 问题描述

给你一个字符串`s`。我们要把这个字符串划分为尽可能多的片段，同一字母最多出现在一个片段中。

返回一个表示每个字符串片段长度的列表。

## 思路分析

### 核心约束

每个字母只能出现在一个片段中。

这意味着：如果一个片段包含字母'a'，那么所有'a'都必须在这个片段中。

### 贪心洞察

对于当前片段，找到其中所有字母最后出现的位置，取最大值作为片段的结束位置。

遍历时，如果到达了当前片段的结束位置，就完成一个片段。

### 两步走

1. 预处理：记录每个字母最后出现的位置
2. 遍历：动态确定每个片段的结束位置

## 代码实现

```javascript
function partitionLabels(s) {
    // 记录每个字母最后出现的位置
    const lastIndex = {};
    for (let i = 0; i < s.length; i++) {
        lastIndex[s[i]] = i;
    }
    
    const result = [];
    let start = 0;
    let end = 0;
    
    for (let i = 0; i < s.length; i++) {
        // 更新当前片段必须延伸到的位置
        end = Math.max(end, lastIndex[s[i]]);
        
        // 到达了当前片段的结束位置
        if (i === end) {
            result.push(end - start + 1);
            start = i + 1;
        }
    }
    
    return result;
}
```

## 正确性证明

### 为什么这样划分是最优的？

**贪心选择性质**：当我们能结束一个片段时就结束（i === end），这样每个片段最短。

如果不立即结束，延迟结束会让片段更长，不会增加片段数量，只会减少。

### 为什么保证正确性？

每个片段包含的字母，其所有出现都在片段内（因为end是这些字母的最大lastIndex）。

所以满足"同一字母最多出现在一个片段中"的约束。

## 特殊情况

### 所有字母相同

```
s = "aaaa"
结果: [4]  // 只能分成一个片段
```

### 所有字母不同

```
s = "abcd"
结果: [1, 1, 1, 1]  // 每个字母单独一个片段
```

## 变种：最少片段数

如果要求最少片段数（而不是最多），答案总是1——整个字符串作为一个片段。

但本题要求"尽可能多的片段"，所以贪心策略是尽早结束每个片段。

## 复杂度分析

**时间复杂度**：O(n)，两次遍历

**空间复杂度**：O(1)，最多26个字母

## 小结

划分字母区间展示了贪心在字符串分割中的应用：
1. **预处理**：记录每个字母的"势力范围"
2. **动态扩展**：遍历时更新片段必须延伸到的位置
3. **及时切割**：一旦到达结束位置就切一刀
