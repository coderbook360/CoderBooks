# 动态规划基础理论

动态规划是解决优化问题的核心方法。

## 什么是动态规划

动态规划（Dynamic Programming，DP）的核心思想：
1. 将大问题分解为子问题
2. 存储子问题的解，避免重复计算
3. 从子问题的解构建原问题的解

## DP vs 分治 vs 贪心

| 方法 | 子问题关系 | 是否存储 | 适用场景 |
|-----|-----------|---------|---------|
| 分治 | 独立 | 否 | 归并排序、快排 |
| 贪心 | 无 | 否 | 局部最优=全局最优 |
| DP | 重叠 | 是 | 最优子结构+重叠子问题 |

## 两个核心条件

### 1. 最优子结构

原问题的最优解包含子问题的最优解。

例如：最短路径问题。A到C的最短路径经过B，则A到B、B到C也必须是最短的。

### 2. 重叠子问题

不同的决策路径会遇到相同的子问题。

例如：斐波那契数列，`fib(5)`需要`fib(4)`和`fib(3)`，而`fib(4)`又需要`fib(3)`。

## 斐波那契数列示例

### 朴素递归（慢）

```javascript
function fib(n) {
    if (n <= 1) return n;
    return fib(n - 1) + fib(n - 2);
}
```

时间O(2^n)，大量重复计算。

### 记忆化递归（自顶向下）

```javascript
function fib(n, memo = {}) {
    if (n <= 1) return n;
    if (memo[n]) return memo[n];
    
    memo[n] = fib(n - 1, memo) + fib(n - 2, memo);
    return memo[n];
}
```

时间O(n)，空间O(n)。

### DP（自底向上）

```javascript
function fib(n) {
    if (n <= 1) return n;
    
    const dp = [0, 1];
    for (let i = 2; i <= n; i++) {
        dp[i] = dp[i - 1] + dp[i - 2];
    }
    return dp[n];
}
```

时间O(n)，空间O(n)。

### 空间优化

```javascript
function fib(n) {
    if (n <= 1) return n;
    
    let prev2 = 0, prev1 = 1;
    for (let i = 2; i <= n; i++) {
        const curr = prev1 + prev2;
        prev2 = prev1;
        prev1 = curr;
    }
    return prev1;
}
```

时间O(n)，空间O(1)。

## 自顶向下 vs 自底向上

### 自顶向下（记忆化递归）
- 从原问题出发，递归分解
- 只计算需要的子问题
- 代码更直观，接近问题描述

### 自底向上（迭代DP）
- 从最小子问题出发，逐步构建
- 计算所有子问题
- 通常空间可以优化

## DP的解题步骤

### 1. 定义状态

`dp[i]`或`dp[i][j]`代表什么？

这是最关键的一步。好的状态定义让问题简单，差的定义让问题复杂。

### 2. 写出状态转移方程

`dp[i]`如何从更小的子问题计算出来？

这是DP的核心，需要分析问题的递推关系。

### 3. 确定初始条件和边界

最小的子问题的解是什么？

### 4. 确定计算顺序

确保计算`dp[i]`时，需要的子问题已经计算过了。

### 5. 返回结果

原问题的答案是`dp[n]`还是`dp[n-1]`还是`max(dp)`？

## 示例：爬楼梯

假设你正在爬楼梯，需要n阶才能到达楼顶。每次可以爬1或2个台阶。有多少种不同的方法？

### 分析

到达第n阶，可以从第n-1阶爬1步，或从第n-2阶爬2步。

### 状态定义

`dp[i]` = 到达第i阶的方法数

### 状态转移

`dp[i] = dp[i-1] + dp[i-2]`

### 初始条件

`dp[0] = 1`（站在地面，一种方法）
`dp[1] = 1`（爬1阶，一种方法）

### 代码

```javascript
function climbStairs(n) {
    if (n <= 2) return n;
    
    let prev2 = 1, prev1 = 2;
    for (let i = 3; i <= n; i++) {
        const curr = prev1 + prev2;
        prev2 = prev1;
        prev1 = curr;
    }
    return prev1;
}
```

## DP问题的分类

### 线性DP
- 爬楼梯、打家劫舍、最大子数组和
- 状态沿一个维度变化

### 二维DP
- 最长公共子序列、编辑距离
- 状态有两个维度

### 区间DP
- 戳气球、矩阵链乘法
- 状态表示一个区间

### 背包DP
- 01背包、完全背包
- 从一组物品中选择

### 树形DP
- 树上的最大独立集
- 在树结构上做DP

## 小结

动态规划的核心：
1. **识别**：问题有最优子结构和重叠子问题
2. **定义**：设计合适的状态
3. **转移**：找出状态之间的关系
4. **优化**：空间压缩

DP需要大量练习才能熟练。接下来我们学习如何定义状态和写出转移方程。
