# 实战：颜色分类

颜色分类是双指针的经典应用，也叫"荷兰国旗问题"。它展示了如何用三指针在一次遍历中完成三路划分。

## 问题描述

给定一个包含红色、白色和蓝色（用0、1、2表示）的数组，原地对它们进行排序，使得相同颜色的元素相邻，且按照红色、白色、蓝色顺序排列。

**示例**：
```
输入：nums = [2,0,2,1,1,0]
输出：[0,0,1,1,2,2]

输入：nums = [2,0,1]
输出：[0,1,2]
```

**要求**：一次遍历，常数空间。

## 思路分析

### 简单方法

1. **计数排序**：统计0、1、2的个数，然后填充。两次遍历。
2. **两次分区**：先把0放前面，再把2放后面。两次遍历。

但题目要求一次遍历。

### 三指针（荷兰国旗）

用三个指针划分数组：

```
[0...p0) = 0    全是0
[p0...i) = 1    全是1
[i...p2] = 未处理
(p2...n) = 2    全是2
```

- `p0`：下一个0应该放的位置
- `p2`：下一个2应该放的位置
- `i`：当前处理的位置

## 完整实现

```javascript
/**
 * @param {number[]} nums
 * @return {void}
 */
function sortColors(nums) {
    let p0 = 0;              // 0的边界
    let p2 = nums.length - 1; // 2的边界
    let i = 0;                // 当前位置
    
    while (i <= p2) {
        if (nums[i] === 0) {
            // 交换到左边0区域
            [nums[i], nums[p0]] = [nums[p0], nums[i]];
            p0++;
            i++;
        } else if (nums[i] === 2) {
            // 交换到右边2区域
            [nums[i], nums[p2]] = [nums[p2], nums[i]];
            p2--;
            // 注意：i不动，因为换来的数还没检查
        } else {
            // 1留在中间
            i++;
        }
    }
}
```

## 执行过程

```
nums = [2, 0, 2, 1, 1, 0]
p0 = 0, p2 = 5, i = 0

step 1: nums[i]=2, 交换nums[0]和nums[5]
  nums = [0, 0, 2, 1, 1, 2]
  p2 = 4, i = 0（不变）

step 2: nums[i]=0, 交换nums[0]和nums[0]
  nums = [0, 0, 2, 1, 1, 2]
  p0 = 1, i = 1

step 3: nums[i]=0, 交换nums[1]和nums[1]
  nums = [0, 0, 2, 1, 1, 2]
  p0 = 2, i = 2

step 4: nums[i]=2, 交换nums[2]和nums[4]
  nums = [0, 0, 1, 1, 2, 2]
  p2 = 3, i = 2（不变）

step 5: nums[i]=1
  i = 3

step 6: nums[i]=1
  i = 4

i = 4 > p2 = 3, 结束

结果：[0, 0, 1, 1, 2, 2]
```

## 关键细节

### 为什么遇到2时i不动？

```javascript
} else if (nums[i] === 2) {
    [nums[i], nums[p2]] = [nums[p2], nums[i]];
    p2--;
    // i不动！
}
```

因为从`p2`换来的数可能是0、1或2，需要再检查一次。

### 为什么遇到0时i可以动？

```javascript
if (nums[i] === 0) {
    [nums[i], nums[p0]] = [nums[p0], nums[i]];
    p0++;
    i++;  // 可以动
}
```

因为`[p0...i)`区间只有1，换来的要么是1（已处理），要么是0（自己和自己交换）。

### 循环条件

```javascript
while (i <= p2)
```

当`i > p2`时，所有元素都已分类完毕。

## 变体：两路划分

如果只有两种颜色（如0和1），更简单：

```javascript
function partition(nums) {
    let p0 = 0;
    for (let i = 0; i < nums.length; i++) {
        if (nums[i] === 0) {
            [nums[i], nums[p0]] = [nums[p0], nums[i]];
            p0++;
        }
    }
}
```

## 复杂度分析

**时间复杂度**：O(n)
- 每个元素最多被访问两次

**空间复杂度**：O(1)
- 只用了三个指针

## 小结

荷兰国旗问题的要点：

1. **三指针划分**：p0管0，p2管2，i负责遍历
2. **遇到0**：换到左边，i和p0都前进
3. **遇到2**：换到右边，p2后退，i不动（要再检查）
4. **遇到1**：不动，i前进

这个技巧在快排的三路划分中很有用。
