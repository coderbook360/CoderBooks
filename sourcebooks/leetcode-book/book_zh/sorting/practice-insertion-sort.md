# 实战：插入排序实现

插入排序的思想类似于整理扑克牌。

## 算法思想

维护一个有序的前缀，每次将下一个元素插入到有序前缀的正确位置。

## 代码实现

```javascript
function insertionSort(arr) {
    const n = arr.length;
    
    for (let i = 1; i < n; i++) {
        const key = arr[i];
        let j = i - 1;
        
        // 把比key大的元素往后移
        while (j >= 0 && arr[j] > key) {
            arr[j + 1] = arr[j];
            j--;
        }
        
        // 插入key
        arr[j + 1] = key;
    }
    
    return arr;
}
```

## 图解

```
初始: [5, 2, 4, 6, 1, 3]
       ^有序

i=1, key=2:
  5 > 2, 5右移
  插入2
  [2, 5, 4, 6, 1, 3]
   ^^有序

i=2, key=4:
  5 > 4, 5右移
  2 < 4, 停止
  插入4
  [2, 4, 5, 6, 1, 3]
   ^^^ 有序

i=3, key=6:
  5 < 6, 停止
  6已在正确位置
  [2, 4, 5, 6, 1, 3]
   ^^^^ 有序

i=4, key=1:
  6,5,4,2都右移
  插入1
  [1, 2, 4, 5, 6, 3]
   ^^^^^ 有序

i=5, key=3:
  6,5,4右移
  2 < 3, 停止
  插入3
  [1, 2, 3, 4, 5, 6]
```

## 二分插入排序

用二分查找找到插入位置，减少比较次数：

```javascript
function binaryInsertionSort(arr) {
    const n = arr.length;
    
    for (let i = 1; i < n; i++) {
        const key = arr[i];
        
        // 二分查找插入位置
        let left = 0, right = i;
        while (left < right) {
            const mid = Math.floor((left + right) / 2);
            if (arr[mid] > key) {
                right = mid;
            } else {
                left = mid + 1;
            }
        }
        
        // 移动元素
        for (let j = i; j > left; j--) {
            arr[j] = arr[j - 1];
        }
        
        arr[left] = key;
    }
    
    return arr;
}
```

二分插入减少了比较次数（O(n log n)），但移动次数不变（O(n²)），所以总时间复杂度仍是O(n²)。

## 复杂度分析

**时间复杂度**：
- 最好：O(n)，数组已排序
- 平均：O(n²)
- 最坏：O(n²)，数组逆序

**空间复杂度**：O(1)

**稳定性**：稳定

## 为什么对部分有序数组高效？

如果数组"几乎有序"，每次插入只需移动少量元素，接近O(n)。

**逆序对**：如果arr[i] > arr[j]且i < j，称为一个逆序对。

插入排序的移动次数 = 逆序对数量。

如果逆序对很少，插入排序非常高效。

## 与选择、冒泡的对比

| 特性 | 冒泡 | 选择 | 插入 |
|-----|------|------|------|
| 最好 | O(n) | O(n²) | O(n) |
| 最坏 | O(n²) | O(n²) | O(n²) |
| 稳定 | 是 | 否 | 是 |
| 自适应 | 是 | 否 | 是 |

插入排序是O(n²)算法中**最实用**的。

## 插入排序的优势

### 1. 对小规模数据高效

常数因子小，对n < 50的数组很快。

### 2. 对部分有序数据高效

逆序对少时接近O(n)。

### 3. 在线算法

可以边接收数据边排序，不需要预先知道所有元素。

### 4. 作为其他算法的子程序

归并排序、快速排序对小数组切换到插入排序。

## 小结

插入排序是简单排序中最实用的：
- 维护有序前缀，逐个插入
- 对小规模和部分有序数据很快
- 稳定、原地、自适应
- 作为高级排序的子程序
