# 实战：有效的括号（进阶）

括号匹配是栈最经典的应用之一。这道题看似简单，实则包含了栈应用的核心思想，是面试中的高频题目。

## 问题描述

给定一个只包括`'('`、`')'`、`'{'`、`'}'`、`'['`、`']'`的字符串`s`，判断字符串是否有效。

有效字符串需满足：
1. 左括号必须用相同类型的右括号闭合
2. 左括号必须以正确的顺序闭合
3. 每个右括号都有一个对应的相同类型的左括号

**示例**：
```
"()"      → true
"()[]{}"  → true
"(]"      → false
"([)]"    → false
"([{}])"  → true
```

## 思路分析

### 为什么用栈？

观察有效括号的特点：**最近的左括号必须与最近的右括号匹配**。

以`([{}])`为例：
```
(  [  {  }  ]  )
↓  ↓  ↓  ↑  ↑  ↑
入 入 入 匹 匹 匹
栈 栈 栈 配 配 配
         {  [  (
```

遇到右括号时，需要匹配**最近**入栈的左括号——这正是栈的LIFO特性。

### 算法流程

1. 遍历字符串的每个字符
2. 如果是左括号，入栈
3. 如果是右括号，检查栈顶是否是对应的左括号
   - 如果匹配，弹出栈顶
   - 如果不匹配，返回false
4. 遍历结束后，栈应该为空

## 完整实现

```javascript
/**
 * @param {string} s
 * @return {boolean}
 */
function isValid(s) {
    // 奇数长度必定无效
    if (s.length % 2 === 1) return false;
    
    const stack = [];
    const map = {
        ')': '(',
        ']': '[',
        '}': '{'
    };
    
    for (const char of s) {
        if (char in map) {
            // 右括号：检查匹配
            if (stack.length === 0 || stack[stack.length - 1] !== map[char]) {
                return false;
            }
            stack.pop();
        } else {
            // 左括号：入栈
            stack.push(char);
        }
    }
    
    // 栈为空说明所有括号都匹配了
    return stack.length === 0;
}
```

## 执行过程图解

### 示例1：`([{}])` → true

```
字符: (  [  {  }  ]  )
栈:  [(] [(,[] [(,[,{] [(,[] [(] []

步骤详解:
1. '(' → 入栈 → stack: ['(']
2. '[' → 入栈 → stack: ['(', '[']
3. '{' → 入栈 → stack: ['(', '[', '{']
4. '}' → 匹配'{' → stack: ['(', '[']
5. ']' → 匹配'[' → stack: ['(']
6. ')' → 匹配'(' → stack: []

栈为空，返回true
```

### 示例2：`([)]` → false

```
字符: (  [  )  ]
栈:  [(] [(,[] 

步骤详解:
1. '(' → 入栈 → stack: ['(']
2. '[' → 入栈 → stack: ['(', '[']
3. ')' → 栈顶是'['，不匹配'(' → 返回false

')'应该匹配'('，但栈顶是'['
这说明括号顺序错误
```

### 示例3：`((` → false

```
字符: (  (
栈:  [(] [(,(]

步骤详解:
1. '(' → 入栈 → stack: ['(']
2. '(' → 入栈 → stack: ['(', '(']

遍历结束，栈不为空
有左括号没有被匹配，返回false
```

### 示例4：`)` → false

```
字符: )
栈:  []

步骤详解:
1. ')' → 栈为空，没有左括号可匹配 → 返回false

右括号找不到对应的左括号
```

## 代码优化技巧

### 技巧1：奇数长度快速判断

```javascript
if (s.length % 2 === 1) return false;
```

有效括号必须成对出现，奇数长度必定无效。

### 技巧2：入栈对应的右括号

可以在遇到左括号时，直接入栈对应的右括号，简化匹配逻辑：

```javascript
function isValid(s) {
    const stack = [];
    
    for (const char of s) {
        switch (char) {
            case '(':
                stack.push(')');
                break;
            case '[':
                stack.push(']');
                break;
            case '{':
                stack.push('}');
                break;
            default:
                // 右括号：直接比较
                if (stack.pop() !== char) {
                    return false;
                }
        }
    }
    
    return stack.length === 0;
}
```

这样右括号只需要和栈顶直接比较，不需要查表。

## 边界情况

| 输入 | 分析 | 结果 |
|------|------|------|
| `""` | 空字符串，栈为空 | true |
| `"("` | 只有左括号，栈不空 | false |
| `")"` | 只有右括号，栈空无法匹配 | false |
| `"()"` | 标准匹配 | true |
| `"()[]{}"` | 多组匹配 | true |
| `"(]"` | 类型不匹配 | false |
| `"([)]"` | 顺序不对 | false |
| `"{[]}"` | 嵌套匹配 | true |

## 复杂度分析

**时间复杂度：O(n)**
- 遍历字符串一次
- 每个字符最多入栈一次、出栈一次

**空间复杂度：O(n)**
- 最坏情况下（全是左括号），栈存储n/2个元素

## 扩展：带其他字符的括号匹配

如果字符串中还有其他字符（如字母、数字），只需忽略非括号字符：

```javascript
function isValidWithOtherChars(s) {
    const stack = [];
    const brackets = { ')': '(', ']': '[', '}': '{' };
    const leftBrackets = new Set(['(', '[', '{']);
    
    for (const char of s) {
        if (leftBrackets.has(char)) {
            stack.push(char);
        } else if (char in brackets) {
            if (stack.pop() !== brackets[char]) {
                return false;
            }
        }
        // 其他字符忽略
    }
    
    return stack.length === 0;
}
```

## 实际应用

括号匹配在实际开发中应用广泛：

1. **代码编辑器**：检查括号是否匹配
2. **编译器**：解析代码块
3. **HTML/XML解析**：匹配开闭标签
4. **数学表达式**：检查括号有效性

## 小结

有效括号问题展示了栈的核心价值：**处理最近相关性问题**。

核心逻辑：
1. 左括号入栈，等待匹配
2. 右括号出栈，验证匹配
3. 最后栈空，全部匹配

**记忆口诀**：左入右出，类型要对，最后要空。
