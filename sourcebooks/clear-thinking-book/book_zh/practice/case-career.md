# 场景9——做出职业选择

## 场景描述

你收到了一个新的工作机会：

```
现状：
- 现在的公司工作稳定，但发展空间有限
- 薪资中等，团队氛围不错

新机会：
- 一家创业公司，薪资涨30%
- 职位是技术负责人，但不确定稳不稳定
- 有期权，但不知道能不能兑现
```

这是**职业选择**场景。

常见的错误做法：
- 凭感觉："感觉这个机会不错"
- 只看钱："涨30%，去！"
- 问别人："你觉得我该不该去？"

正确的做法是**系统分析，做出适合自己的决策**。

## CPERF应用

### C - Capture 捕获

**第一步：明确决策问题**

```
问题：是否接受这个新机会？
本质：在稳定和发展之间做选择
```

**第二步：收集信息**

```
关于现在的工作：
- 优点：稳定、团队好、离家近
- 缺点：天花板明显、薪资涨幅有限
- 发展：预计2-3年能升一级，之后就很难了

关于新机会：
- 优点：职位高、薪资涨30%、有期权
- 缺点：创业公司不稳定、工作强度可能大
- 发展：如果公司做成了，发展空间大

关于自己：
- 职业阶段：工作5年，想要更大的发展
- 风险承受：有一定积蓄，能承受半年失业
- 家庭情况：未婚，负担不重
```

**第三步：补充关键信息**

```
还需要了解的：
1. 创业公司的业务情况和融资情况
2. 具体的职责和汇报关系
3. 期权的具体条款
4. 工作强度和文化
```

去面试时主动了解这些信息：

```
了解到：
- 公司已完成B轮融资，资金够用2年
- 技术负责人直接向CEO汇报
- 期权有4年归属期，第一年不兑现
- 工作强度：996比较常见
```

### P - Process 处理

**列出决策维度和权重**：

根据自己的情况，确定什么对自己最重要：

```
维度和权重：
1. 职业发展：40%（最看重）
2. 薪资收入：25%
3. 工作稳定性：20%
4. 工作生活平衡：15%
```

**对比分析**：

```markdown
### 现在的工作

职业发展（权重40%）：★★☆☆☆
- 天花板明显，2-3年后很难再升

薪资收入（权重25%）：★★★☆☆
- 中等水平，涨幅有限

工作稳定性（权重20%）：★★★★★
- 非常稳定

工作生活平衡（权重15%）：★★★★☆
- 加班不多，离家近

### 新机会

职业发展（权重40%）：★★★★★
- 技术负责人，能力提升大
- 如果公司发展好，空间很大

薪资收入（权重25%）：★★★★☆
- 涨30%
- 期权有风险但也有想象空间

工作稳定性（权重20%）：★★☆☆☆
- 创业公司，有一定风险
- 但B轮融资完成，2年内应该没问题

工作生活平衡（权重15%）：★★☆☆☆
- 996，工作强度大
```

**计算对比**：

```
现在的工作：2*0.4 + 3*0.25 + 5*0.2 + 4*0.15 = 3.15
新机会：    5*0.4 + 4*0.25 + 2*0.2 + 2*0.15 = 3.70
```

**分析结果**：新机会得分更高，但差距不大。

**进一步分析**：

```
如果新机会成功的话：
- 职业发展：天花板大大提升
- 收入：期权可能值很多

如果新机会失败的话：
- 损失：最多失业半年
- 但能力提升是实在的，再找工作有竞争力

考虑到自己的情况：
- 5年经验，想要更大发展
- 风险承受能力OK
- 负担不重
```

**决策**：倾向于接受新机会。

### E - Express 表达

**整理自己的决策逻辑**：

```
我决定接受新机会，理由如下：

1. 职业发展是我现阶段最看重的
   现在的工作天花板明显，新机会能打开空间

2. 风险在可承受范围内
   创业公司有风险，但B轮融资完成，2年内应该OK
   即使失败，我也能承受半年失业

3. 这个阶段适合拼一下
   未婚，负担不重，可以承担一定风险
   再过几年可能就不敢拼了

4. 最坏情况也不差
   即使公司失败，我在技术负责人岗位的经历是有价值的
   再找工作有竞争力
```

### R - Review 复盘

**决策后持续验证**：

```
入职1个月后：
- 工作内容符合预期
- 强度确实大，但在承受范围
- 公司氛围还不错

入职半年后：
- 能力提升明显
- 带团队的经验积累很多
- 公司业务有增长

持续关注：
- 公司业务发展
- 自己的成长速度
- 是否需要调整方向
```

**假设验证**：

```
假设1："技术负责人岗位能大幅提升能力"
→ 验证中，目前确实学到很多

假设2："公司2年内应该稳定"
→ 需要持续观察

假设3："即使失败，经历也有价值"
→ 目前来看成立
```

### F - Form 固化

**职业选择的框架**：

```markdown
## 职业选择决策框架

### 第一步：明确决策问题
- 我面临的选择是什么？
- 本质上是在什么之间做选择？

### 第二步：收集信息
- 现有选项的优缺点
- 新选项的优缺点
- 自己的情况（阶段、风险承受、家庭）

### 第三步：确定维度和权重
- 对我来说什么最重要？
- 各个维度的权重是多少？

### 第四步：对比分析
- 各选项在各维度的得分
- 综合得分对比

### 第五步：风险分析
- 最好情况是什么？
- 最坏情况是什么？
- 最坏情况能承受吗？

### 第六步：做出决策
- 综合各方面，倾向哪个？
- 决策逻辑是什么？

### 第七步：持续验证
- 决策后持续观察
- 假设是否成立？
- 是否需要调整？
```

## 关键要点

### 要点1：明确自己最看重什么

```
不同的人看重的不一样：

有人最看重稳定
有人最看重发展
有人最看重收入
有人最看重工作生活平衡

没有对错，但要知道自己最看重什么。
```

### 要点2：收集充分的信息

```
决策的质量取决于信息的质量。

不要只听对方说，要主动了解：
- 公司的真实情况
- 岗位的真实职责
- 可能的风险

信息不够就不要急着做决定。
```

### 要点3：考虑最坏情况

```
不要只想好的一面，要问：

如果失败了会怎样？
最坏情况我能承受吗？
有没有退路？

能承受最坏情况，才能安心做选择。
```

### 要点4：决策后持续验证

```
决策不是终点。

入职后持续观察：
- 实际情况和预期一致吗？
- 假设成立吗？
- 需要调整吗？

如果发现假设不成立，要有调整的勇气。
```

### 要点5：考虑自己的阶段

```
不同阶段适合不同的选择：

年轻、负担轻：可以多拼一拼，承担一些风险
中年、负担重：稳定性可能更重要

选择要符合自己的阶段。
```

## 小结

职业选择的核心方法：
1. 明确自己最看重什么
2. 收集充分的信息
3. 系统对比分析
4. 考虑最坏情况
5. 决策后持续验证

记住：**没有完美的选择，只有适合自己的选择。**
