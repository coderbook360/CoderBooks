# 实战：移掉K位数字

这道题展示了单调栈在"构造最优序列"中的应用——在保持相对顺序的前提下，构造字典序最小的数字。

## 问题描述

给你一个以字符串表示的非负整数`num`和一个整数`k`，移除这个数中的`k`位数字，使得剩下的数字最小。请以字符串形式返回这个最小的数字。

**示例**：
```
输入: num = "1432219", k = 3
输出: "1219"
解释: 移除4, 3, 2三个数字，得到最小的数1219

输入: num = "10200", k = 1
输出: "200"
解释: 移除1，注意结果不能有前导零

输入: num = "10", k = 2
输出: "0"
解释: 移除所有数字，结果是0
```

## 思路分析

### 贪心策略

为了让数字最小，高位应该尽可能小。

观察`"1432219"`：
- 从左往右扫描
- `1`比`4`小，保留
- `4`比`3`大，应该移除4，这样3就到了高位
- 继续：`3`比`2`大，移除3
- `2`比`2`相等，保留
- `2`比`1`大，移除2
- 已移除3位，结束

**规律**：当高位数字比低位数字大时，移除高位可以让数字变小。

### 单调递增栈

维护一个递增的栈：
- 新数字比栈顶小时，弹出栈顶（相当于移除）
- 最多弹出k次

## 完整实现

```javascript
/**
 * @param {string} num
 * @param {number} k
 * @return {string}
 */
function removeKdigits(num, k) {
    const stack = [];
    
    for (const digit of num) {
        // 贪心：弹出栈顶比当前大的数字
        while (k > 0 && stack.length && digit < stack[stack.length - 1]) {
            stack.pop();
            k--;
        }
        stack.push(digit);
    }
    
    // 如果还没移除够k位（如递增序列），从末尾移除
    while (k > 0) {
        stack.pop();
        k--;
    }
    
    // 去除前导零
    const result = stack.join('').replace(/^0+/, '');
    
    return result || '0';
}
```

## 执行过程图解

以`num = "1432219"`, `k = 3`为例：

```

  2 = 2，不弹出
  入栈

digit='1':
  1 < 2，弹出2，k--

```

## 处理递增序列

如果数字是递增的，如`"12345"`，k=2：
- 遍历时不会弹出任何数字
- 最后从末尾移除：`"123"`

```javascript
// 关键代码
while (k > 0) {
    stack.pop();
    k--;
}
```

## 处理前导零

移除数字后可能产生前导零，如`"10200"`移除`1`后是`"0200"`。

```javascript
// 去除前导零
const result = stack.join('').replace(/^0+/, '');
return result || '0';  // 全是0时返回"0"
```

## 另一种写法：直接处理前导零

```javascript
function removeKdigits(num, k) {
    const stack = [];
    
    for (const digit of num) {
        while (k > 0 && stack.length && digit < stack[stack.length - 1]) {
            stack.pop();
            k--;
        }
        // 不将前导零入栈
        if (stack.length || digit !== '0') {
            stack.push(digit);
        }
    }
    
    // 从末尾移除
    while (k > 0 && stack.length) {
        stack.pop();
        k--;
    }
    
    return stack.join('') || '0';
}
```

## 边界情况

| 输入 | 分析 | 结果 |
|------|------|------|
| `"10", k=1` | 移除1 | "0" |
| `"10", k=2` | 全部移除 | "0" |
| `"112", k=1` | 移除最后的2 | "11" |
| `"100", k=1` | 移除1，前导零 | "0" |
| `"9", k=1` | 单字符移除 | "0" |

## 复杂度分析

**时间复杂度：O(n)**
- 每个数字最多入栈出栈各一次

**空间复杂度：O(n)**
- 栈最多存储n个数字

## 与"去除重复字母"的联系

两道题都用单调栈构造"最优序列"：

| 特点 | 移掉K位数字 | 去除重复字母 |
|------|-------------|--------------|
| 目标 | 移除k位，数值最小 | 去重，字典序最小 |
| 栈类型 | 递增栈 | 递增栈 |
| 弹出条件 | 当前 < 栈顶 且 k > 0 | 当前 < 栈顶 且 栈顶后面还有 |
| 额外约束 | 移除数量限制 | 每个字符只能出现一次 |

## 小结

移掉K位数字的核心：

1. **贪心策略**：让高位尽可能小
2. **单调递增栈**：遇到更小的数字就弹出栈顶
3. **移除限制**：最多弹出k次
4. **后处理**：递增序列从末尾移除，处理前导零

这道题是"用单调栈构造最优序列"的典型案例，思路可以推广到类似的"移除/选择元素使结果最优"的问题。
