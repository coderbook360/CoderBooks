# 序言

## 为什么写这本书

渲染器是 Vue3 将虚拟 DOM 转换为真实 DOM 的核心引擎。它包含了框架中最复杂的算法——Diff 算法。

作为一名前端开发者，你可能听说过 Virtual DOM 和 Diff 算法，但你是否真正理解：

- VNode 虚拟节点是什么结构？
- 为什么需要 Virtual DOM 这层抽象？
- Diff 算法是如何高效对比新旧节点的？
- 最长递增子序列算法如何优化移动操作？
- Scheduler 调度器如何实现批量更新？

这些问题的答案，隐藏在 Vue 渲染器的设计之中。本书将带你深入 `@vue/runtime-dom` 的源码，全面理解渲染器的实现原理。

## 本书特色

本书采用 **"设计思想 + 源码解析 + Mini 实现"** 三合一的架构。

### 第一部分：设计思想

- Virtual DOM 的诞生与演进
- Diff 算法的设计思路
- 自定义渲染器的设计

### 第二部分：源码解析

逐行解读 Vue 渲染器核心源码：

- VNode 创建
- patch 核心流程
- Diff 算法实现
- Scheduler 调度器

### 第三部分：Mini 实现

从零实现一个功能完整的 Mini 渲染器。

## 目标读者

本书面向具有 **3-5 年前端开发经验** 的开发者。

## 你将收获

- ✅ 深入理解 Virtual DOM 的设计原理
- ✅ 掌握 Diff 算法的优化策略
- ✅ 独立实现一个 Mini 渲染器
- ✅ 解决项目中的渲染性能问题
