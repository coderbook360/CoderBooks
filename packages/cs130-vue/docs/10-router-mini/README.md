# 从零实现 Mini Vue Router

> 手写路由系统，深刻理解前端路由原理 - L4-10 实现书

## 📖 书籍定位

本书是 **Vue Router 的实现书**（L4-10），通过从零手写一个功能完整的 Mini Router，帮助读者深刻理解路由系统的设计与实现。

## 🎯 核心内容

- **History 管理**：Hash 和 History 模式实现
- **路由匹配**：路径解析、正则匹配、参数提取
- **导航系统**：push/replace/go 实现
- **导航守卫**：守卫系统设计与实现
- **内置组件**：RouterView 和 RouterLink 实现
