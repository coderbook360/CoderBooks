# 从零实现 Mini Vue Router (L4-10)

## 书籍概述

本书带你从零开始实现一个功能完整的 Mini Vue Router，通过实践深入理解 Vue Router 的核心实现原理。

## 核心内容

- **路由匹配器**：实现路径匹配与参数解析
- **History 管理**：实现 Hash 和 HTML5 History 模式
- **导航守卫**：实现完整的守卫执行流程
- **组件系统**：实现 RouterView 和 RouterLink
- **嵌套路由**：实现嵌套路由的完整支持

## 学习目标

- ✅ 理解路由匹配的算法实现
- ✅ 掌握 History API 的使用
- ✅ 实现导航守卫的执行流程
- ✅ 理解路由组件的渲染机制
- ✅ 完成一个可用的 Mini Router
