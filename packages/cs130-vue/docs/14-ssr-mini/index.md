# 从零实现 Mini Vue SSR (L4-14)

## 书籍概述

本书带你从零开始实现一个功能完整的 Mini SSR 框架，通过实践深入理解 Vue SSR 的核心实现原理。

## 核心内容

- **服务端渲染**：实现 renderToString
- **流式渲染**：实现 renderToStream
- **组件渲染**：实现组件的服务端渲染
- **客户端 Hydration**：实现客户端水合
- **状态传递**：实现状态序列化与恢复

## 学习目标

- ✅ 理解同构渲染的实现原理
- ✅ 掌握字符串渲染的算法
- ✅ 实现流式输出的优化
- ✅ 理解水合过程的细节
- ✅ 完成一个可用的 Mini SSR
