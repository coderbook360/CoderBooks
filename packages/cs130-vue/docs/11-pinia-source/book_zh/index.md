# 序言

## 为什么要深入 Pinia 源码？

Pinia 是 Vue 3 官方推荐的状态管理库，它摒弃了 Vuex 的复杂概念，提供了更简洁、更符合 Composition API 风格的状态管理方案。

但你是否真正理解：

- Pinia 相比 Vuex 做了哪些设计改进？
- Setup Store 和 Options Store 有什么区别？
- Store 的响应式是如何实现的？
- 插件系统是如何工作的？
- 如何设计一个现代化的状态管理库？

这些问题的答案，就隐藏在 Pinia 的源码中。
