# 序言

## 为什么写这本书

编译器是 Vue3 中最"神奇"的部分。它将我们编写的模板语法转换为高效的渲染函数，这个过程对大多数开发者来说是一个黑盒。

作为一名前端开发者，你可能每天都在使用模板语法，但你是否真正理解：

- 模板字符串是如何被解析的？
- AST 抽象语法树是什么结构？
- Transform 阶段做了什么优化？
- 最终的渲染函数代码是如何生成的？
- 静态提升和补丁标记是如何提升性能的？

这些问题的答案，隐藏在 Vue 编译器的设计之中。本书将带你深入 `@vue/compiler-core` 的源码，全面理解编译器的实现原理。

## 本书特色

本书采用 **"设计思想 + 源码解析 + Mini 实现"** 三合一的架构。

### 第一部分：设计思想

- 编译原理基础概念
- Vue3 编译器的设计目标
- 编译时优化策略

### 第二部分：源码解析

逐行解读 Vue 编译器核心源码：

- baseParse 词法分析
- AST 节点结构
- Transform 转换流程
- codegen 代码生成

### 第三部分：Mini 实现

从零实现一个功能完整的 Mini 编译器。

## 目标读者

本书面向具有 **3-5 年前端开发经验** 的开发者。

## 你将收获

- ✅ 深入理解编译器的工作原理
- ✅ 读懂 Vue 编译器源码的核心实现
- ✅ 独立实现一个 Mini 编译器
- ✅ 开发自定义编译器插件
