# 序言

## 为什么写这本书

组件是现代前端框架的核心抽象。在 Vue3 中，组件系统经历了重大升级，Composition API 的引入让组件的组织方式更加灵活和强大。

作为一名前端开发者，你可能已经熟练编写 Vue3 组件，但你是否真正理解：

- 组件实例是如何创建的？内部有哪些关键属性？
- Props 和 Emits 是如何被处理和验证的？
- Slots 插槽的作用域是如何实现的？
- setup 函数在什么时机执行？返回值如何与模板关联？
- Teleport、Suspense、KeepAlive 这些特殊组件是如何工作的？

这些问题的答案，隐藏在 Vue3 组件系统的设计之中。本书将带你深入 `@vue/runtime-core` 的源码，全面理解组件系统的每一个细节。

## 本书特色

本书采用 **"设计思想 + 源码解析"** 二合一的架构：

### 第一部分：设计思想

理解"为什么"比知道"怎么做"更重要。

- 为什么需要组件化？
- Options API 和 Composition API 各有什么优势？
- 组件通信有哪些模式？如何选择？
- 生命周期为什么这样设计？

### 第二部分：源码解析

逐行解读 `@vue/runtime-core` 的组件相关源码：

- createApp 应用创建流程
- defineComponent 组件定义
- 组件实例创建与初始化
- Props、Emits、Slots 处理
- setup 函数执行
- 生命周期钩子实现
- provide/inject 依赖注入
- 特殊组件（Teleport、Suspense、KeepAlive）
