# 序言

## 为什么要深入 Vue Router 源码？

Vue Router 是 Vue 生态系统中不可或缺的一部分。作为官方的路由管理库，它为单页应用提供了完整的路由解决方案。

但你是否真正理解：

- 前端路由是如何工作的？Hash 模式和 History 模式有什么区别？
- 路由匹配算法是如何实现的？动态路由如何高效匹配？
- 导航守卫的执行流程是怎样的？为什么需要 next 函数？
- RouterView 和 RouterLink 是如何实现的？
- 如何设计一个企业级的路由系统？

这些问题的答案，就隐藏在 Vue Router 的源码中。

## 本书特色

本书是 **L4-9 Vue Router 源码深度解析书**，采用 **"设计思想 + 源码解析"** 的结构：

### 第一部分：设计思想

- 前端路由的演进历史
- Hash、History、Memory 三种模式的设计原理
- 路由匹配、导航守卫等核心概念
- 与 React Router 的设计对比

### 第二部分：源码解析

逐行解读 Vue Router 核心源码：

- History 模式的实现机制
- 路由匹配器的算法细节
- 导航系统的完整流程
- 组件的渲染逻辑
