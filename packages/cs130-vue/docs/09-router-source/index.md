# Vue Router 源码深度解析

> 深入理解前端路由的设计与实现 - L4-9 源码解析书

## 📖 书籍定位

本书是 **Vue Router 的源码解析书**（L4-9），深入解析 Vue Router 的设计思想和核心实现，帮助读者理解前端路由系统的完整架构。

## 🎯 核心内容

- **设计思想**：Hash/History/Memory 模式原理
- **路由匹配**：路径解析、正则匹配、参数提取
- **导航系统**：push/replace/go 的实现细节
- **导航守卫**：守卫执行流程与异步处理
- **内置组件**：RouterView 和 RouterLink 源码
