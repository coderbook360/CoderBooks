# Vue Router 设计与实现

## 书籍概述

本书深入解析 Vue Router 4 的设计思想与源码实现，带你从理解原理到手写实现，彻底掌握前端路由系统的核心机制。

## 核心内容

- **History API**：浏览器历史记录管理与导航
- **路由匹配**：路径解析、正则匹配与路由树
- **导航守卫**：全局、路由、组件级守卫机制
- **路由视图**：RouterView 与 RouterLink 实现
- **滚动行为**：页面滚动位置管理
- **Mini 实现**：从零实现完整的路由系统

## 书籍结构

### 第一部分：设计思想（约 18 章）

深入理解路由系统的设计理念与权衡：

- 前端路由发展历史
- Hash 模式 vs History 模式
- 路由匹配算法设计
- 导航守卫设计思想
- 路由与状态管理集成
- 嵌套路由与布局设计

### 第二部分：源码解析（约 25 章）

逐行解读 vue-router 核心源码：

- 整体架构与模块划分
- createRouter 创建流程
- History 实现（createWebHistory/createWebHashHistory）
- 路由匹配器（Matcher）实现
- 路由解析与路径处理
- 导航守卫执行流程
- RouterView 组件实现
- RouterLink 组件实现
- 滚动行为处理
- 路由懒加载实现

### 第三部分：Mini 实现（约 17 章）

从零实现一个完整的 Mini 路由系统：

- 项目架构设计与接口定义
- 实现 History 管理
- 实现路由匹配器
- 实现导航守卫
- 实现 RouterView 与 RouterLink
- 单元测试与验证
- 性能优化与扩展

## 技术栈

- **核心框架**: Vue 3.4+
- **路由库**: vue-router 4.x
- **语言**: TypeScript
- **关键 API**: History API、URL、正则表达式
