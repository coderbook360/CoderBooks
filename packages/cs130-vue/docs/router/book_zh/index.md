# 序言

## 为什么写这本书

路由是现代单页应用（SPA）的基础设施。Vue Router 作为 Vue 官方的路由解决方案，提供了强大而灵活的导航能力。

作为一名前端开发者，你可能已经熟练使用 Vue Router 配置路由、设置守卫，但你是否真正理解：

- History 模式和 Hash 模式的本质区别是什么？
- 路由匹配是如何从路径解析到组件的？
- 导航守卫的执行顺序为什么是这样的？
- router-view 是如何知道应该渲染哪个组件的？
- 嵌套路由是如何工作的？

这些问题的答案，隐藏在 Vue Router 的设计之中。本书将带你深入 vue-router 的源码，全面理解前端路由的实现原理。

## 本书特色

本书采用 **"设计思想 + 源码解析 + Mini 实现"** 三合一的架构：

### 第一部分：设计思想

- 前端路由的发展历程
- Hash 模式 vs History 模式的权衡
- 路由匹配算法的设计
- 导航守卫的设计思路

### 第二部分：源码解析

逐行解读 vue-router 的核心源码：

- createRouter 创建流程
- History 实现
- 路由匹配器
- 导航守卫执行
- RouterView 和 RouterLink 组件

### 第三部分：Mini 实现

从零实现一个功能完整的 Mini 路由系统。

## 目标读者

本书面向具有 **3-5 年前端开发经验** 的开发者。

## 你将收获

- ✅ 深入理解前端路由的设计原理
- ✅ 读懂 vue-router 源码的核心实现
- ✅ 独立实现一个 Mini 路由系统
- ✅ 设计复杂的路由架构方案
