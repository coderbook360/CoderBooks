# Vue SSR 源码深度解析 (L4-13)

## 书籍概述

本书深入解析 Vue3 服务端渲染的设计思想与源码实现，帮你彻底理解同构渲染的核心机制。

## 核心内容

- **设计思想**：SSR、CSR、SSG 的对比与选择
- **同构渲染**：服务端与客户端渲染的统一
- **renderToString**：字符串渲染的实现原理
- **renderToStream**：流式渲染的优化策略
- **Hydration**：客户端水合的完整流程
- **状态传递**：服务端到客户端的状态同步
