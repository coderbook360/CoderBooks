# 序言

## 为什么写这本书

> 纸上得来终觉浅，绝知此事要躬行。

本书将带你从零开始，手写一个功能完整的 Mini 组件系统。

## 本书定位

本书是 [《Vue3 组件系统源码深度解析》](/component/) 的配套实践书籍。

## 你将实现什么

### 应用系统

- `createApp`：应用创建
- `app.mount`：应用挂载
- `app.use`：插件安装

### 组件实例

- 组件实例创建
- 组件实例属性
- setup 函数执行

### 属性与事件

- Props 初始化与验证
- Emits 事件机制
- v-model 实现

### 插槽系统

- 基础插槽
- 作用域插槽

### 生命周期

- 钩子注册机制
- 生命周期执行

### 依赖注入

- provide 实现
- inject 实现
