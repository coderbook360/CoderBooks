# Vue3 核心设计总览

## 书籍概述

本书是 Vue3 源码级系列的总览指南，对比分析响应式、组件、渲染器、编译器四大核心模块的设计理念，以及 Router、Pinia、SSR 的架构方案，帮助你建立 Vue3 生态系统的全局视野。

## 核心内容

- **架构设计哲学**：Vue3 的设计理念与核心价值
- **四大核心对比**：响应式、组件、渲染器、编译器的设计选择
- **路由与状态管理**：Router 和 Pinia 的方案对比
- **SSR 同构渲染**：服务端渲染的架构设计
- **技术选型方法论**：如何做出正确的架构决策

## 为什么需要这本总览书？

在深入学习各个模块的源码之前，你需要先理解：

- 为什么 Vue3 选择 Proxy 而不是 Object.defineProperty？
- 为什么需要 Composition API？Options API 有什么问题？
- 为什么编译器要做静态提升和补丁标记优化？
- 为什么需要 Block Tree 和 PatchFlags？
- 各个模块之间如何协作？设计边界在哪里？

这些问题的答案，就是本书要解答的核心内容。
