# 序言

## 为什么写这本书

服务端渲染（SSR）是提升首屏性能和 SEO 的重要技术。Vue3 提供了完整的 SSR 支持，但对大多数开发者来说，SSR 的工作原理仍然是个谜。

作为一名前端开发者，你可能听说过 SSR 和 SSG，但你是否真正理解：

- 同构渲染是如何实现的？
- renderToString 和 renderToStream 有什么区别？
- 水合（Hydration）是什么过程？
- 为什么会出现 Hydration 不匹配的问题？
- 流式渲染如何提升性能？

这些问题的答案，隐藏在 Vue SSR 的设计之中。本书将带你深入 `@vue/server-renderer` 的源码，全面理解服务端渲染的实现原理。

## 本书特色

本书采用 **"设计思想 + 源码解析 + Mini 实现"** 三合一的架构。

### 第一部分：设计思想

- SSR 的诞生背景与价值
- 同构应用的设计挑战
- 流式渲染的优势

### 第二部分：源码解析

逐行解读 Vue SSR 核心源码：

- renderToString 实现
- renderToStream 实现
- Hydration 流程

### 第三部分：Mini 实现

从零实现一个功能完整的 Mini SSR 方案。

## 目标读者

本书面向具有 **3-5 年前端开发经验** 的开发者。

## 你将收获

- ✅ 深入理解 SSR 的设计原理
- ✅ 读懂 Vue SSR 源码的核心实现
- ✅ 独立实现一个 Mini SSR 方案
- ✅ 设计高性能的 SSR/SSG 架构
