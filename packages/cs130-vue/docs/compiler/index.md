# Vue Compiler 源码深度解析

## 书籍概述

本书深入解析 Vue3 编译器的设计思想与源码实现，带你彻底读懂模板编译的完整流程。

## 核心内容

- **词法分析**：模板字符串的 Token 化处理
- **语法分析**：AST 抽象语法树的构建
- **Transform 转换**：AST 节点的转换与优化
- **代码生成**：从 AST 到渲染函数的生成
- **编译优化**：静态提升、补丁标记等优化策略

## 书籍结构

### 第一部分：设计思想（约 20 章）

深入理解编译器的设计理念与权衡：

- 编译原理基础概念
- 模板编译 vs JSX 编译
- Vue3 编译器设计目标
- 编译时优化策略
- 运行时与编译时的权衡
- SFC 单文件组件编译流程

### 第二部分：源码解析（约 30 章）

逐行解读 Vue 编译器核心源码：

- 整体架构与模块划分
- baseParse 词法分析
- AST 节点类型定义
- parseElement 元素解析
- parseInterpolation 插值解析
- parseText 文本解析
- Transform 转换流程
- transformElement 元素转换
- transformExpression 表达式转换
- codegen 代码生成
- 静态提升优化
- 补丁标记（PatchFlags）
- SFC 编译流程
