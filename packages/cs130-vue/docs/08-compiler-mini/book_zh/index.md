# 序言

## 为什么写这本书

> 纸上得来终觉浅，绝知此事要躬行。

本书将带你从零开始，手写一个功能完整的 Mini 编译器。

## 本书定位

本书是 [《Vue Compiler 源码深度解析》](/compiler/) 的配套实践书籍。

## 你将实现什么

### 词法分析

- Token 定义
- 词法分析器基础

### 语法解析

- 元素解析
- 属性解析
- 插值解析
- AST 节点结构

### AST 转换

- Transform 框架
- 元素转换
- v-if 转换
- v-for 转换
- 表达式转换

### 代码生成

- 代码生成器
- 元素生成
- 静态提升
- 补丁标记

## 目标读者

- ✅ 已阅读 [《Vue Compiler 源码深度解析》](/compiler/)
- ✅ 熟悉 TypeScript 基础语法
- ✅ 了解编译原理基础概念
