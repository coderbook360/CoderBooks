# 从零实现 Mini Vue Component

## 书籍概述

本书带你从零开始，手写一个功能完整的 Mini 组件系统，通过亲手实现彻底掌握 Vue3 组件原理。

## 核心内容

- **createApp 实现**：应用创建与挂载流程
- **组件实例实现**：实例创建、属性初始化
- **Props/Emits 实现**：属性验证与事件机制
- **Slots 实现**：插槽与作用域插槽
- **生命周期实现**：钩子注册与执行
- **provide/inject 实现**：依赖注入机制
