# 从零实现 Mini Vue Compiler

## 书籍概述

本书带你从零开始，手写一个功能完整的 Mini 编译器，通过亲手实现彻底掌握 Vue3 编译器原理。

## 核心内容

- **词法分析实现**：Token 定义与词法分析器
- **语法解析实现**：元素解析、属性解析、插值解析
- **AST 实现**：抽象语法树节点结构
- **Transform 实现**：元素转换、v-if/v-for 转换、表达式转换
- **代码生成实现**：Codegen、元素生成、静态提升
