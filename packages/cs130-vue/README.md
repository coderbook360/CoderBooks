# cs130-vue

> Vue3 生态系统完整学习系列 - 从零到精通的系统化学习路径

## 📚 系列书籍

本项目包含 7 本聚焦 Vue3 源码级别的专业书籍：

- 🎯 **响应式系统** - Proxy、依赖收集、effect 机制
- 🧩 **组件系统** - 生命周期、渲染机制、Composition API
- 🛣️ **路由系统** - History API、路由匹配、导航守卫
- 🔧 **编译器系统** - 模板编译、AST 转换、代码生成
- 🎨 **渲染器系统** - 虚拟 DOM、Diff 算法、Patch 策略
- 📦 **状态管理** - Pinia 设计原理与源码实现
- 🌐 **服务端渲染** - SSR 架构与同构应用开发

## 🚀 快速开始

### 安装依赖

```bash
pnpm install
```

### 本地开发

```bash
pnpm run docs:dev
```

### 构建部署

```bash
pnpm run docs:build
```
